## Ch1：族谱

```
视角：沈氏族谱阁轮值抄录弟子（外部视角——主角不登场，但全章关于主角）

场景：族谱阁。抄录弟子在此轮值两年——每天用朱笔在族谱上勾销被逐出者的名字。

核心发现：族谱卷尾全是旁支页——被朱笔勾销的名字全在这里。族中长老十二人分属六房，被勾销的无一人长辈在这十二人中。

老书吏（修笔管三十年）的回应："你手上还有几个名要勾？""三个。""勾完。笔管修好了。继续抄。"

悬念：主脉那一页——沈安条目下，本该有一道朱笔。三年前宣读过逐出令，但念完时他父亲沈怀远起身斟茶——半盏茶泼在了令纸上。墨迹晕开。执事说"重抄一份。明日再议。"三年过去。明日还没到。朱笔没落。

抄录弟子竹册末页余墨小字："族规对人有别。一茶之隔。"

基调：讽刺观察，不悲凉。"一勾一划，比过年写对联还利索。"
```

## Ch2：别院的早上

```
视角：沈安（主角首次出场）

核心：
- 沈安靠着廊下老青石柱——柱子上被他的后背磨出了浅痕
- 今天的阳光角度比昨天偏了半指——秋分过后的太阳往南走
- 晨钟晚了半刻——撞钟的老孙昨晚又喝了酒（儿子在矿上捡了块石头不是灵石但他觉得是）——这件事全城只有三个人知道。沈安是第四个——老孙撞钟时走路的步数多了一步
- 沈安感应到族谱阁方向有一道朱笔落下了——今天又有一个名字被勾销
- 沈怀远站在廊下。极短的对话。一张折叠的纸放在台阶上——正面是族报训诫，背面一行手写小字："我明日去矿脉。你自己吃饭。"
- 沈安把纸翻到背面——指尖感应沈怀远留在纸上的道痕余温。余温是凉的。说明他写完这行字之后没有立刻离开

不写：风、风大、风凉等景物渲染
```

## Ch2B：沈怀远的记忆（在Ch2与Ch3之间插入）

```
视角：沈怀远

核心：
- 沈怀远离开别院后没有回戒律堂——去了后院。后院老槐树上有一道十七年前刻的痕
- 那年沈安三岁。他第一次发现儿子不对劲：三岁的沈安蹲在院子角落指着地上说"这下面的脉不对"
- 沈怀远顺他的手指往下挖了三尺——什么都没有
- 第二天暴雨，山洪冲开院墙地基——地基下露出一道道痕矿石脉
- 沈安蹲在那里手指不动——好像在等什么
- 后来他知道了：那不是三岁孩子在玩泥巴。那个孩子已经在看地底的道痕了
- 但他不能让人知道。戒律长老的儿子如果被发现生来就能感知地底道痕——这不是天赋，是造假的把柄
- 沈怀远在槐树上刻了一道痕。那天晚上他做了两件事：帮儿子把挖过的坑填平，和自己说"我什么都不知道"

点题："满级"——三岁就能看地底道痕。"重修"——不是天生，他有前世。沈怀远不能问——他只能等儿子自己告诉他
```

## Ch3：严父

```
视角：沈怀远

场景：族谱阁

核心：
- 沈怀远离开别院后没有回戒律堂。他去了族谱阁——想确认沈安那页还在不在。虽然他每天都知道答案
- 他翻开族谱主脉册——沈安的条目还在。没有朱笔。三年前那盏茶泼湿的不是令纸，是族会上所有人盯着的他的脸
- 他把族谱放回原位。老书吏在角落里修笔管——头没有抬
- 沈怀远走到门口停了一步。老书吏的声音从背后传过来："沈长老——今日添了新墨。黑的还是朱的。"沈怀远没回头。"黑的。我来修笔。"
- 揭示：沈怀远每天来族谱阁做同一件事——看一眼，确认墨没有变色，然后离开。老书吏知道。老书吏从不问。这个阁里修了三十年笔管的人知道族里所有不该问的事

结尾：老书吏把刚修好的笔放在窗台上。光照在朱墨上——墨是刚磨的。今天没有人用
```

## Ch4-5：沈青萝出场 + 日轨

```
Ch4：沈青萝首次出场——沈氏年轻一辈第一天才，窥痕境·明痕。沈崇武大长老的孙女。她在族中大会上第一次注意到沈安——不是因为他有什么表现，恰恰因为他没有任何表现。全族大会所有人都正襟危坐——只有沈安靠着柱子，眼睛半合。他在"听"，但不是在听人说话——他脊骨贴着石柱在感应天痕之脉此刻的枢点律动。沈青萝开始暗中观察沈安——她画了第一张日轨图

Ch5：沈青萝继续观察——沈安的活动轨迹规律到让她觉得"太规律了等于太不规律"。她画出第二张日轨图。沈安第一天就察觉到了她的跟踪——他没有阻止。一个不跟他说话的女人提供了免费的第三方观测数据——她的日轨可以用来反向验证他是否在不经意间泄漏了不该有的道痕气机

不写：风。沈安不打哈欠。沈青萝不脸红心动。
```

### 沈安的6项行为（每一项都有天痕之脉的目的）

| 旁人误解 | 实际在做 | 为什么非这样不可 |
|---------|---------|-----------------|
| 晒太阳/换柱子 | 脊骨贴着石柱，被动感知天痕之脉枢点时辰律动 | 主动探测=释放道痕=在天痕之脉留痕=巡天鉴可见。被动只用肉身+经验——不留痕 |
| 计算阳光角度精确到半指 | 天痕之脉气机随日照变化——用最低成本做道痕气象学 | 前世预留的观测方法：不用刻痕境就能做——物理定位替代道痕探测 |
| 每一项刚好踩在及格线 | 精确的暴露风险评估——描多了可能共鸣，描少了被退货失去观测环境 | 及格线是当前最优解。不是懒——是精确到发丝的风险管理 |
| 看蚂蚁行军 | 行军蚁天然沿地脉道痕迁徙——是活的道痕气机显示仪 | 观测成本为零。不需释放任何道痕——蚂蚁替你做了道痕测绘 |
| 不阻止沈青萝跟踪 | 沈青萝的日轨是免费的第三方隐匿精度验证 | 她的记录越详细→他对自己的泄漏判断越准。不跟他说话的女人=免费分析工具 |
| 教沈小禾道痕 | 对照实验——验证道痕体系底层设计在普通人身上的实际响应 | 教描红不需封印力量——不会共鸣。比被随时打断找人的时间成本低 |

### 镜头规则

每段至少一个视觉锚点（光线角度/空间位置/物体细节/距离感）。没有视觉锚点的段落=不合格。

合格：
- "柱子上被他的后背磨出了浅痕。"
- "老书吏在角落里修笔管——头没有抬。"
- "光照在朱墨上——墨是刚磨的。今天没有人用。"

不合格：
- 全段对话
- 全段内心独白
- 抽象概念堆砌
