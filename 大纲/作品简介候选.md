# 作品简介·候选

## 版本一

```
道痕修炼体系是他前世发明的。重生后，他决定每一项刚好及格就行。

不是懒——是他太清楚每一项的精确及格线在哪。多描一道都可能被天痕之脉捕捉，而在天痕之脉的另一端，有人正在掘地三尺地找他。不是来找他报仇的。只是想当面问清楚一件事。但他不知道。他以为对方要来揍他。

所以他躲。躲在一根柱子后面，用最省力的方式做完最精确的事。

所有人都以为他在偷懒。没有人知道他连打一个哈欠的力气都舍不得浪费——不是舍不得力气，是舍不得分心。

他最大的敌人不是追他的人。是他自己的舌头。憋了一千年的话，这一世逮谁说谁。总有一天会出事。
```

## 版本二

```
前世：道痕体系的创造者，站在修炼之路的尽头。

今生：窥痕境初痕。三年没动过。不是不能突破——是每一次突破都可能暴露自己的位置。在看不见的天痕之脉另一端，有一个追了他很久的人。

他给自己定了一条规矩：每一项刚好及格。

多一分不用，少一分不行。描红描到刚好不重描，晨练做到刚好不补训。全宗找不出一条宗规说他错了——因为没有任何一条规矩写着"必须多做"。他像一个站在及格线上的数学家，用最低的能耗做最精准的事。

所有人都觉得他在偷懒。

没有人知道他已经把这个世界的底层脉理算了一辈子。
```

## 版本三

```
他是个很怕麻烦的人。

怕到什么程度呢——怕到为了不被别人感激，用执痕级别的天痕之脉操作去帮一个小姑娘躲掉一场她不想要的"天才培养计划"。怕到明知道有人每天跟踪研究他，但因为对方不跟他说话，他决定随她去。

他前世走到过修炼之路的终点。这一世从头再来——不是重走，是重修。他自己设计的体系，他比任何人都知道怎么走最快。但最快就意味着暴露。所以他选了一条没人能看懂的路线：每一项刚好及格。

及格是比满分更难的精确度。满分只要用力，及格需要知道停在哪里。

他停了一辈子。总有一天停不下来。
```
