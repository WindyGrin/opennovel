# 爽点规划

> 项目：满级重修，每一项刚好及格就行｜题材：修仙喜剧｜创建：2026-05-02
> 
> 本书爽点遵从《创作基调约束》第四条：爽点来自"别人怎么想这件事"而非"这件事本身"。主角不出手的胜利，远比主角碾压的胜利好笑。

## 核心卖点

- 玄霄巅峰大佬假装渡劫失败逃到苍渊，追他的不是仇人——是被老婆拿捏的妻管严兄弟。兄弟其实只想问明白一件事，但主角以为兄弟要来揍他。这个误会持续了四百年。
- 全书最高爽点不是"谁赢了"——是"谁的脸被打得最响但不疼，因为当事人不知道脸被打过"。

## 爽点分层

### 一、即时小爽点（每章至少1-2个）

**类型定义**：读完一章后读者嘴角上扬一次以上的瞬间。不依赖连续阅读。

| 类型 | 示例 | 原理 |
|---|---|---|
| **嘴瓢爽** | 沈安在最不该开口的时刻说了一句大实话——"爹，你求人的姿势不太对。"全场死寂。 | 他把所有人正在想但不敢说的东西说出来了。 |
| **反套路打脸** | 对手蓄力十秒大招→沈安在看台下的猫→对手自己踩裂了年久失修的石板→人从裂缝掉了下去→裁判判负，理由是"离开擂台"。沈安收回目光——猫走了。 | 传统爽文是主角碾压。本书是主角不动手，天意帮主角解决。 |
| **社死揭穿** | 沈安的"秘密"被揭穿了——但不是实力被揭穿，是嘴瓢说了拭痕的梦话:"爹……我要吃酒……"在场所有人:这人八成有病。 | 揭穿的方向永远是社死，不是装逼。 |
| **勤奋地狱** | 砥砺宗全员苦练——沈安在角落晒太阳——忽然道痕突破了。大师兄跪在地上:"你怎么做到的!!!"沈安:"呃——晒着晒着就——" | 越努力越心酸。越躺平越突破。 |

### 二、卷末大爽点（每卷1个）

**类型定义**：一卷读完后的那个"必须马上看下一章"的瞬间。

| 卷 | 大爽点 | 为什么爽 |
|---|---|---|
| 1 | 沈安为救沈守拙，拈起一枚石子弹中阵法节点。全场死寂。他不是在展现实力——他是在救一个人的命。救完假装什么都没发生。沈守拙散场后只说了"谢谢"。 | 第一次窥见主角的真正分量。不是力量的分量——是"舍与藏"的分量。 |
| 2 | 云隐宗内鬼长老被当众揭发→但揭发者不是沈安——是沈守拙的妹妹。沈安从头到尾只做了一件事:给了她一个证据。她在台前对质，他在台下晒太阳。 | 主角帮助别人成为英雄——这是比"主角成为英雄"更高级的爽感。 |
| 3 | 厉淮在虚海一间破茶馆里对沈安说:"下次走远点。别让我下属看见。"沈安:"你有本事叫你下属别找。"厉淮:"我没有。所以我喝茶。喝完回去继续排查。" | 两个被迫的人之间的同病相怜。这不是传统的"征服对手"——这是"对手变成了你的同谋"。 |
| 4 | 沈安在苍渊秘境找到拭痕。拭痕在梦里叫了一声:"爹……你来了……我赢了吗……"沈安没有回答。他拿来一坛酒——是他三百年前封在封印旁边的那坛醉仙引。他在封印边坐了一整夜。 | 全书最安静的爽点。不是赢——是一句你欠了三百年的道歉终于说了出来，虽然它睡着了没听见。 |
| 5 | 萧长河和沈安在茶馆里对坐。萧长河问的不是"你为什么跑"——是"我那坛酒还在不在"。沈安沉默了很久。他把杯子推过去。两人都没说话。茶是凉的——早就凉了。 | 四百年的追逃在这一卷结束。不是一场大战——是两个人终于愿意坐下来喝一杯凉了的茶。 |
| 6 | 苏锦年放弃"法则化"——不是因为被打败，是因为萧长河说了三个字。拭痕完全苏醒，笔身亮成青色太阳。沈安:"今天不打赌。今天爹带你去打一场。"拭痕:"打完请我喝酒。" | 全书最大的爽点:不是谁赢了谁输了——是一个女人发现自己恐惧了一辈子的事其实不存在，一个男人打赢了一局不必打了，一支笔喝到了它等了三百年的那杯酒。 |

### 三、反套路爽点类型（全书独有）

| 类型 | 经典场景 | 与传统爽文的区别 |
|---|---|---|
| **勤奋者心碎** | 砥砺宗弟子质问沈安:"你为什么突破了!!!"沈安:"可能——可能是粥?" | 传统爽文是用主角的努力碾压别人。本书是用别人的努力反衬主角的"躺赢"。 |
| **社死式成名** | 沈青萝在家族会议逐条报告沈安的每日活动轨迹。全族:"他们果然在谈恋爱。"沈安:"我没有——"沈青萝:"你怎么证明你没有。"沈安张了张嘴，发现解释起来太麻烦——不解释了。 | 被以为在装逼的时候在搞研究。被以为在搞研究的时候其实在谈恋爱。被以为在谈恋爱的时候其实什么都没干。 |
| **反派被迫营业** | 厉淮给"找到了重要线索"的下属颁发奖章。手在抖。下属:"执事太感动了。" | 反派不想当反派。反派想下场喝茶。 |
| **感情线全宇宙嗑CP** | 温小鹿和沈安在图书馆两时辰不说一个字→全宗管这叫"认真交往"。温小鹿:"那是他们闲。" | 两个人之间的关系越淡，全世界的想象越丰富。 |
| **反派自爆** | 玉华阁:我们从不道歉——我们换话题。 | 不是主角打败了她们——是她们自己终于说累了。 |

## 密度目标

| 节奏单位 | 目标 |
|---|---|
| 每章 | 至少1-2个"嘴角上扬"的小爽点 |
| 每10章 | 至少1个"拍大腿"的中爽点 |
| 每卷 | 1个"必须追下一卷"的大爽点 |
| 全书 | 6个大爽点+1个终极爽点(萧长河三字+拭痕苏醒) |

## 爽点红线

- ❌ 禁止主角通过碾压对手获得爽感——他的爽感来自"别人强行想碾压他然后被天道意外反制"
- ❌ 禁止任何"众人倒吸凉气→认出主角→跪下"的桥段——本书的围观者反应永远是"？""他在干嘛""这人是不是病得不轻"
- ❌ 禁止"隐藏实力被当众揭穿然后所有人震惊"——如果有人揭穿他的实力，揭穿方向必须是社死（嘴瓢/眉心血迹/酒嗅）

## 景物描写红线

- ❌ **禁止用风/雨/天气做情绪渲染**——除非事件本身依赖天气（如台风毁屋）。"风大""风凉""微风拂面"这类填充式景物描写一律删除
- ❌ **禁止景物烘托情绪**——让事实和人物反应说话，不借景抒情。讽刺优先于抒情

## 写作自检规则

### 镜头自检

每章初稿完成后，逐段问自己：**读者能从这段文字中看到什么？**

如果一段文字全是对话、内心活动或抽象描述——没有可视的光线、没有空间中的位置、没有物体的物理细节——该段标记为待修改。

合格视觉锚点举例：
- "柱子上的浅痕比去年深了一线。"
- "石屑落在族谱阁的案角上，积了薄薄一层。"
- "阳光从他膝盖移到了脚踝。"

### 写前读词

每章写作前，根据本章可能涉及的关键场景，查阅 `设定集/核心设定/设定词典.md` 第七节"禁忌词库"中对应分类的禁用词汇：

| 本章场景 | 需避开的关键词 |
|---------|-------------|
| 刻碑/祠院/族规执行 | 档案/文书/表格/存档/流程 |
| 修炼/道痕/天痕之脉 | 网络/节点/数据/频率/备份 |
| 考核/宗门规制 | 成绩单/考核表/评分/数据对比 |
| 人际关系/道契 | 协议/机制/PTSD/系统/预警 |
| 势力/经济往来 | 百分比/覆盖/数据库/商务模式 |

## 沈安的行为语言：创造者的重修

> 沈安不是在偷懒。他是道痕体系的创造者正在重修自己设计的路。每一步都精确到暴露风险的及格线上。以下每一件事——旁人看来是懒散——实际上没有一件事是浪费时间。

| 旁人误解 | 实际在做 | 为什么非这样不可 |
|---------|---------|-----------------|
| 晒太阳/换柱子 | 脊骨贴着石柱，被动感知天痕之脉枢点的时辰律动 | 主动探测=释放道痕=在天痕之脉留痕=巡天鉴可见。被动感知只用肉身+经验——不留痕 |
| 计算阳光角度精确到半指 | 天痕之脉气机随日照角度变化——用最低成本做道痕气象学 | 前世预留的观测方法：不用刻痕境的感知就能做——靠物理定位替代道痕探测 |
| 每一项刚好踩在及格线 | 精确的暴露风险评估——描多了可能共鸣，描少了被退货失去观测环境 | 及格线是当前最优解。不是懒——是精确到发丝的风险管理 |
| 看蚂蚁行军 | 行军蚁天然沿地脉道痕迁徙——是活的道痕气机显示仪 | 观测成本为零。不需要释放任何道痕——蚂蚁替你做了道痕测绘 |
| 不阻止沈青萝跟踪 | 沈青萝的日轨记录是免费的第三方隐匿精度验证工具 | 她的记录越详细——他对自己是否泄漏气机的判断越准。一个不跟他说话的女人=免费外部分析工具 |
| 教沈小禾道痕 | 对照实验——一个普通人在纯粹描红路径上的自然成长，验证道痕体系底层设计 | 教描红不需动用封印力量——不会共鸣。且每周固定教一次=比被随时打断找人的时间成本低。他算过 |

### 使用规则
1. 沈安的任何"懒散"行为必须有道痕层面的隐藏目的——这书的笑点不是"他懒"，是"所有人以为他懒，其实他一直在一刻不停地观测和计算"
2. 同一类型的"旁人误解→道痕真相"不连续使用超过2次
3. 旁人的误解不能是愚蠢——必须是在他们那个境界合理但境界不够所以理解不了
