# 首次创作者风险声明与规避指南

> **声明**：本项目维护者为首次创作小说的业余创作者。
> 本文档旨在**诚实、透明**地声明创作中存在的各类风险，
> 以及已采取的应对措施，供共创者和读者参考。

---

## 风险一：AI 辅助创作的法律与平台风险

### 事实

- 作者无传统网文写作经验，正文由 AI 工具辅助生成
- AI 平台：本框架所调用的 LLM 模型
- 人类角色：世界观设计、角色设定、情节大纲、内容审核、人工润色

### 法律风险

- **著作权归属不确定**：中国《著作权法》现行对 AI 生成内容的版权归属无明确规定。司法实践倾向于保护"体现人类智力创作"的作品，纯 AI 输出可能不受版权保护。
- **2025 年 AI 生成内容标识新规**：国家网信办等部门对 AI 生成内容的标识要求趋严，平台可能要求标注 AI 参与程度。

### 平台风险

- **番茄小说 AI 内容政策不明**：番茄小说对 AI 辅助创作的态度尚未完全明确，存在签约被拒或作品被下架的风险。
- **同行举报风险**：其他作者可能以"AI 生成内容"为由举报，导致平台审查。

### 已采取的规避措施

1. 采用"**人类主导 + AI 辅助**"模式，非全自动生成：
   - 人类创作：总纲、卷纲、章节大纲、世界观、角色设定、情节走向、审核润色
   - AI 辅助：行文输出、设定一致性检查、上下文检索
2. 在 README 和章节开头诚实声明 AI 参与程度
3. 保留人工创作过程记录（大纲手稿、设定讨论、审核日志）作为"人类智力投入"的证据
4. 签约前主动向番茄平台咨询 AI 辅助作品政策
5. 每章进行人工审核和润色，确保非纯 AI 输出

## 风险二：番茄平台排他性风险

### 事实

- 本书正文计划在**番茄小说**平台签约发布
- 大纲、设定集、番外在 GitHub 开源
- 正文滞后 2 卷后选择性公开

### 风险

- 番茄小说签约条款通常要求**全网首发且独家**，公开大纲/设定可能被认定为"部分发布"
- GitHub 上的开源内容若被第三方搬运至其他平台，可能触发番茄的独家违约判定

### 已采取的规避措施

1. **正文不在 GitHub 公开** — 仅在番茄独家发布
2. **大纲/设定开源但不构成"正文发布"** — 大纲和设定属于创作构思层面的分享，不等同于作品发表
3. **应急预案**：若收到番茄平台下架警告：
   - 立即从 GitHub 移除正文相关文件
   - 保留大纲、设定集、番外的开源状态
   - 与番茄平台沟通确认合规边界
4. **首发时间戳**：番茄发布时间优先于 GitHub 公开时间，作为首发权利证明

## 风险三：共创版权纠纷风险

### 事实

- 本项目接受 GitHub 社区贡献（设定建议、情节讨论、素材提供）
- 共创者不获得经济报酬，仅享有署名权

### 风险

- 中国《著作权法》规定合作作品的版权为共有
- 若共创者后续主张版权分成，可能产生法律纠纷

### 已采取的规避措施

1. **CLA（贡献者许可协议）** — 所有贡献者在 CONTRIBUTING.md 中明确同意：
   - 财产性著作权授权给项目维护者
   - 确认无经济报酬要求
   - 保留署名权
2. **双轨署名** — 在 AUTHORS.md 和章节末尾标注贡献者，满足署名权
3. **透明声明** — 贡献条款在 CONTRIBUTING.md 中公开，接受监督

## 风险四：现实题材改编的合规风险

### 事实

- 本书定位为"根据现实改编，讽刺、鞭策现实生活"
- 鼓励共创者将现实案例整合进小说

### 风险

- **名誉权/隐私权**：若小说中的人物、事件与现实中可识别的个人/事件过于相似，可能被指控侵权
- **内容审查**：讽刺现实题材在网文平台面临更严格的审核
- **公众舆论**：若读者将虚构内容对号入座，可能引发争议

### 已采取的规避措施

1. **强制三层虚构化** — 所有现实案例必须经过三层处理后方可纳入：
   - 第一层：时代/地域架空
   - 第二层：人物身份转换
   - 第三层：核心矛盾提炼
2. **创作基调约束** — 见 `设定集/创作基调约束.md`
3. **免责声明** — 在 README 中声明"本作品为虚构创作，如有雷同纯属巧合"
4. **自我审查机制** — 每章发布前进行内容敏感性检查

## 风险五：模仿者竞争风险

### 事实

- 大纲、设定集、世界观在 GitHub 开源
- 该框架可能被他人用于创作类似作品

### 风险

- 设定被他人"借鉴"，创作出市场竞品
- 开源许可限制可能被忽视或绕过

### 已采取的规避措施

1. **CC BY-NC-SA 许可** — 创作内容禁止商业使用，模仿者无法合法发布至付费平台
2. **首发时间戳** — 番茄发布时间 + Git commit 历史 = 可追溯的原创证明
3. **品牌壁垒** — 独特的世界观和角色绑定是不可替代的竞争壁垒
4. **内容滞后** — 正文滞后 2 卷公开，模仿者无法同步跟进
5. **法律路径** — 若模仿者违反 CC 许可进行商业使用，保留法律追究权利

## 风险六：文笔与质量问题

### 事实

- 作者无网文写作经验，正文主要依赖 AI 输出
- AI 输出可能出现逻辑断裂、人设不一致等问题

### 风险

- 作品质量可能不达读者预期
- 番茄平台读者差评、弃读率高等

### 已采取的规避措施

1. 使用本框架内置的**审查管线**（一致性、连贯性、OOC、节奏等）
2. 每章进行人工审核和润色
3. 在 README 中**诚实声明**作者背景，管理读者预期
