# 毒点标签格式与速查

## 文件架构

```
设定集/毒点标签库.md              ← 速查索引（审查必加载，~25行）
设定集/毒点标签库/<标签名>.md      ← 完整条目（按需加载，含档位锚点+案例）
```

## 速查索引格式

`设定集/毒点标签库.md` 为 Markdown 表格：

```markdown
| 类别 | 标签 | 毒度基准 | 触发条件（摘要） | 惯性 |
|------|------|:------:|------|:----:|
| 文笔 | 句式复读机 | 7 | 同句式出现≥5次 | 1 |
```

- `惯性`: 连续触发章数。每次复用 +1，中断归零。≥3 时审查报告触发惯性警告。

## 子文件条目格式

`设定集/毒点标签库/<标签名>.md`：

```markdown
# 标签名

- 类别: [文笔/情节/角色/叙事/结构/设定]
- 触发条件: [完整触发条件]
- 毒度基准: [3-10]
- 档位锚点:
  | 轻(X) | [轻档判断标准] |
  | 中(Y) | [中档判断标准] |
  | 重(Z) | [重档判断标准] |
- 首次记录: [第X章]
- 最近出现: [第Y章]
- 惯性计数: [N]

## 典型案例
> 原文: "[引用]"
> 批评: "[一句话]"
```

## 档位说明

审查时对每个崩点判定严重度档位，修正基线分：

| 档位 | 修正 | 含义 |
|------|:---:|------|
| 轻 | 基线 - 2 | 刚触及触发线，影响局部 |
| 中 | 基线 ± 0 | 典型触发，符合标签定义 |
| 重 | 基线 + 2 | 远超触发线，影响整章体验 |

调整后分值钳制 [1, 10]。

## 更新规则

| 情况 | 操作 |
|------|------|
| 发现新崩点类型 | 创建新子文件 + 同步追加索引表行 |
| 已有标签被复用 | 更新子文件的"惯性计数"和"最近出现"，同步索引表惯性列 |
| 标签30章未复用 | 追加 `#已免疫` 标记 |
| 新案例更优 | 替换子文件的典型案例 |

## 综合毒度公式

```
综合毒度 = (Σ 调整后崩点分²) / (Σ 调整后崩点分) × 密度系数
```

- 二次均值：高分崩点权重天然更重
- 密度系数：崩点 1-4→×1.00, 5-6→×1.05, 7+→×1.10
- 0崩点章节不计算综合毒度
