# 崩点标签格式与速查

## 标签文件位置

`设定集/崩点标签库.md`

## 标签条目格式

```markdown
## [标签名]
- 类别: [文笔/情节/角色/叙事/结构/设定]
- 触发条件: [什么情况下触发]
- 毒度基准: [3-10]
- 首次记录: [第X章]
- 最近出现: [第Y章]
- 典型案例:
  > 原文: "[引用]"
  > 批评: "[一句话]"
```

## 更新规则

| 情况 | 操作 |
|------|------|
| 发现新崩点类型 | 追加新标签条目 |
| 已有标签被复用 | 更新"最近出现"和"典型案例" |
| 标签30章未复用 | 追加 `#已免疫` 标记 |

## 崩点标签速查表

| 类别 | 标签 | 毒度基准 |
|------|------|---------|
| 文笔 | 句式复读机、匠气太重、废话连篇、描写虚浮 | 6-8 |
| 情节 | 拐弯走冲突、节奏掉崖、爽点注水、钩子断裂 | 7-9 |
| 角色 | 谜语人装逼、工具人上线、情绪跳水 | 6-8 |
| 叙事 | 旁白抢戏、知识点播、感知展览 | 5-7 |
| 结构 | 开头太空、结尾太虚、过渡硬切 | 5-6 |
| 设定 | 战力崩坏、引用错误、过度暴露 | 8-10 |
