# 三段式报告模板

## 毒度标尺

| 毒度 | 含义 | 典型表述 |
|------|------|---------|
| 9-10 | 致命 | "不改发出去就是掉收" |
| 7-8 | 重创 | "读者在这里会切走" |
| 5-6 | 明显 | "及格了，但经不起第二遍读" |
| 3-4 | 微瑕 | "不影响追读，但累积多了就是天花板" |
| 1-2 | 吹毛求疵 | "我就想找点东西骂你" |

## 报告骨架

```
═══════════════════════════════════════
       毒舌审查 · 第X章「标题」
═══════════════════════════════════════

📊 综合毒度: X.X/10  ｜  崩点: N个  ｜  模式: 要害打击/full

───────────────────────────────────────
🔪 崩点一：[标签名]  毒度 X/10
───────────────────────────────────────

① 开骂 / 读者视角
[第二人称"你"，一句话打穿]
读完这段我的真实反应：[一句话]

② 解剖 / 编辑视角
定位: 第X段
根因: [写法/设计/认知]
证据:
> [引用原文1]
> [引用原文2]
[为什么这崩了——原理层面分析]

[若涉及设定:]
> 📖 设定参考: [设定名称]（见 设定集/XX/XX.md:行号）
> 你的理解: [写成什么样]
> 实际设定: [设定原文是什么]

③ 改法 / 作者视角
演示:
> [2-3句重写示范]

要点: [不是只改这一句，是整个习惯要改什么]

───────────────────────────────────────
🔪 崩点二：[标签名]  毒度 X/10
───────────────────────────────────────
[同样三段式]

───────────────────────────────────────
📋 次生问题速览（一句带过，不改也行）
───────────────────────────────────────
- [第X段]: [一句话]
- [第Y段]: [一句话]
═══════════════════════════════════════
```

## 保存后输出

```
✅ 毒舌审查完成: 第{CHAPTER}章
   - 综合毒度: {SCORE}/10
   - 崩点: {N}个
   - 报告: 审查报告/第{CHAPTER}章毒舌审查.md
   - 标签库: {UPDATED/NEW/NONE}
```
