# 三段式报告模板

## 毒度标尺

| 毒度 | 含义 | 典型表述 |
|------|------|---------|
| 9-10 | 致命 | "不改的话，写到这儿必崩" |
| 7-8 | 重创 | "按这个大纲写，读者到卷中会大量流失" |
| 5-6 | 明显 | "大纲及格了，但经不起第二遍审视" |
| 3-4 | 微瑕 | "不影响大局，但累积多了就是结构债" |
| 1-2 | 吹毛求疵 | "我就想找点东西骂你" |

## 综合毒度公式

```
综合毒度 = (Σ 调整后崩点分²) / (Σ 调整后崩点分) × 密度系数
```

- 二次均值：高分崩点权重天然更重
- 密度系数：崩点 1-4→×1.00, 5-6→×1.05, 7+→×1.10
- 0崩点不计算综合毒度

```
✅ 大纲毒舌审查完成: [总纲/第N卷]

   - 崩点: N个 (轻:N 中:N 重:N)
   - 报告: 审查报告/大纲/<目标>毒舌审查.md
   - 标签库: 更新N个/新建N个
```
