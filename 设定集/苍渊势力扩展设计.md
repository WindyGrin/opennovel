# 苍渊势力扩展设计（修订版·二）

## 一、青石学宫与"女天才"事件

### 青石学宫

- **位置**:苍渊东域·青石城郊外。与沈氏家族所在的青石城相距仅半日脚程。原本是一座废弃的道痕试炼场——沈氏先祖在三百年前捐赠给城池作为"公学"之用。最初的青石学宫是一个朴素的地方:不收束脩，不问出身，来者即可旁听。青石城里的散修子弟、小商贩的子女、无门无派的年轻人，都能在这学最基础的道痕启蒙。沈怀远年轻时曾在这里授课——他第一次见到铁骊也是在青石学宫的讲堂里，那时两人都还是学生。

多年后学宫日渐凋零。公学的性质决定了它永远缺钱。两位老教习退休后无人接替，藏书阁的卷轴被借走后再也没还回来。为了活下去，学宫不再"来着不拒"——它开始包装"优材生"来吸引富户子弟的束脩。包装成功了，可以开大价钱招下一批。包装失败了，就换一个。

- **修炼方式**:最基础的道痕启蒙。没有核心功法——学宫的性质决定了它只是"学前班"而非宗门。大部分学生从学宫毕业后去往各大宗门继续深造。学宫在此过程中收取宗门"推荐费"——这构成了它的主要收入来源。
- **关键人物**:
  - 宫主（姓周）——第三代宫主。接手时学宫账面已赤字三年。他的每一个决定都不是出于恶意——是出于"学宫明天就要关门了"的恐惧。
  - 裘教习（名待取）——学宫仅存的最后一位正式教习。六十来岁，教了半辈子道痕启蒙，连地阶都没摸到边。周宫主的所有事他都看在眼里——他选择了不看。

### 女天才"纪秋萤"

纪秋萤是青石学宫的一名普通弟子。父母是在青石城里开杂货铺的散修，请不起宗门的束脩，只能送她到学宫来旁听。她的道痕试炼成绩大致是——中等偏下。在学宫里算不上最差，但跟"天才"二字之间的距离约等于从苍渊到玄霄再回来。

周宫主需要一个天才。青石学宫已经连续两年没有产出过"优材生"了——没有可以对外宣传的成功案例，下一季的富户招生就要黄。他在学宫里挑了两个人选。第一个表现太差无人相信。第二个就是纪秋萤。理由很实际:她面容清秀，谈吐尚可，符合大众对"少年天才"的基本预期。

周宫主的计划并不复杂:在一年一度的"青石城公开道痕考核"中，由裘教习在幕后协助纪秋萤完成三场考核——用教习级别的手段配合弟子级别的表演，看起来就像"一个普通少女凭借惊人的道痕悟性在考核中突然绽放"。

计划出现了第一个意外:纪秋萤的真正水平比周宫主预估的还要低。裘教习不得不加大了幕后协助的力度，导致考核的结果看起来不是"她爆发了"而是"她不可能做到这些"——但坐在台下的青石城观众不懂这些。他们只看到一个少女在考核台上大放异彩。

玉华阁正是在这一刻出现的。

### 玉华阁接手

玉华阁的阁主之一（玉阁之主，专门负责对外话语权）在青石城出差时恰好路过了这场考核。她看到的不是一场作弊——她看到的是一个完美的舆论产品。一个少女。平凡出身。在大庭广众之下证明自己。只要角度对，这就是"苍渊最年轻的女性道痕天才"。

玉华阁的舆论机器在全速运转三个月后产生了以下结果:

- 纪秋萤被授予"青石之光"称号
- 玉华阁出资为她翻修了青石城的旧学宫(牌匾换成了新的，上面刻着玉华阁的徽记)
- 各大宗门收到了玉华阁措辞优美的推荐函——推荐纪秋萤进入宗门深造
- 青石城街头的茶馆里每一个人都在谈论这个少女

云隐宗是最早表现出兴趣的大宗门之一。玉华阁的函件写得极其精妙——不说是邀请函，说的是"贵宗如有关注青年道痕天才之雅趣，纪秋萤或为可造之材"。"或为"这个词用得恰到好处，既表达了推荐的诚意又给自己留足了回旋余地。

云隐宗的回应同样精妙:"宗内近期考核排期满额，烦请贵弟子稍待时日。届时定以公平考核相迎。"翻译:我们现在没空理你。但话不能说死——万一她真不是假的呢。

### 沈安的角度

沈安在教习考核现场，是偶然路过的——那天他在青石城找一个卖桂花糕的铺子。铺子没找到，考核看了十分钟，花糕也不找了。

他的道痕感知告诉他——考核台上不是一个人在操作。有第二道道痕介入的痕迹，极为熟练，是个老手。但他没有说。不是懒得说——说了没人信，还得解释"你怎么看出来的"——太麻烦。

沈青萝全程在场。她不是路过——她是来观摩考核的，顺便找她在大庭广众之下公开批评了沈安之后被沈怀远私下说了一顿，心情烦躁，需要一个地方散心。考核看到一半她的表情从烦躁变成了困惑。她凑到沈安旁边，极轻声地说:"教习人数对吗。"

沈安:"嗯。"

沈青萝:"你知道多久了。"

沈安:"比你早大约一炷香。"

沈青萝:"你为什么不说。"

沈安:"说了又怎样。"

沈青萝沉默了片刻，然后把手上的考核评分表折了起来——她本来打算作为沈氏的评审代表给一个正面评价的。她把笔收回了袖子里。

当天晚上，她开始了新一份"沈安观察记录"——这一份的开头只有一句话:"此人对真相的容忍度高于一切。"

### 揭穿

揭穿不是来自外部调查——是来自青石学宫内部。裘教习的良心在折磨了他三个月之后找到了出口:他私下找到了自己在云隐宗的旧同窗，在那位同窗递给他的一杯浓茶里哭了一整个下午。

周宫主收到裘教习失踪的消息时表情很平静。他把学宫的门关上，在椅子上坐了一会儿。然后拿起纸笔，开始给自己写下一季的招生计划书。他心里有一个声音在说"这不对"，但他压住了。学宫不能关。这是他唯一能说服自己的理由。

裘教习的爆料在苍渊宗门圈内缓慢传播——不像舆论爆炸，更像水波扩散。到第三个月时，所有大宗门的核心层都知道了这件事。没有一家公开表态。因为表态了就是站队，站队了就要解释为什么之前不去查——太麻烦。

玉华阁在第四个月发了一份新的函件:内容不谈纪秋萤，谈"青年女性修士在苍渊道痕启蒙阶段的困境与结构性不公"。把话题从"她是不是造假"转移到"凭什么只针对她"。文案依然优美，逻辑依然无懈可击。这是玉华阁的看家本事——她们从不道歉，她们只换话题。

云隐宗从未公开回应过自己当初的"承诺考核"。但柳师兄后来对沈安说了一句——"师兄们其实知道她是假的。但这话不能从我嘴里说出去。"

### 事件余波

- 纪秋萤回到了青石城杂货铺。她父亲什么也没问。母亲每天多给她加一个菜。
- 周宫主继续当他的宫主。学宫没有关门——裘教习走了，但新来的年轻教习没有见证过这件事，所以学宫可以继续假装什么都没发生。
- 沈青萝在第三个月的某天，把一张从学宫废墟里捡到的泛黄试卷叠好收进了抽屉——那是纪秋萤在作弊事件以前参加的一次真实考核。分数不高。但每一道题都是她自己写的。沈青萝没有告诉任何人她留了这张。她自己也不知道为什么。
- 沈安对此事件的最终评价:"所有问题都是她身边的成年人搞出来的。"然后继续找他的桂花糕。找到了，味道一般。

## 二、飞舟之争:砥砺宗 vs 星汉阁

### 砥砺宗的飞舟体系

砥砺宗掌控着苍渊东域最大的飞舟运输网络——"铁脊航线"。从铁脊山脉起始，贯穿苍渊东部全境，每年运输的货物和修士数不胜数。这是砥砺宗最大的收入来源，也是铁氏家族控制宗门的核心经济命脉。

飞舟的制造方法是铁氏的不传之秘:**《砥砺船术》**——将《砥砺心经》的理念转化为飞舟动力。船体由铁脊山脉特产的"脊钢"锻造，在锻造过程中由数位修炼《砥砺心经》的长老同时向船体内打入道痕刻印。打进去的越多，船越坚固、运输越稳定。一艘旗舰飞舟通常需要三位长老同时协作锻造。

飞舟不"飞"——是靠船体内的道痕刻印与苍渊大气中的残留法则产生共振，在极低的高度缓缓滑行。安全、耗能少、但慢。任何一艘飞舟都需要数位专业"技师"（修炼《砥砺船术》的弟子）随船维护道痕刻印。

铁氏控制航线的原理极其简单:所有旗舰的关键道痕只有铁氏嫡系能维护。散修能学会驾驶、能学会装货、能学会基本的保养——但如果船体道痕出了问题，必须请铁氏的人来修。一张说大不大说小不小的关系网，把所有人拴在了铁氏的船坞上。

### 星汉阁:来自天空的反叛者

星汉阁的创始人叫**陆川**。曾经是砥砺宗的弟子——三十年前，他向铁氏长老会提交了一份提案:用"小型化道痕"替代传统巨舰。铁氏的回复不是"拒绝"——是"你负责的甲板擦完了吗。"

陆川没有擦甲板。他离开了砥砺宗，用十年时间在苍渊极西的荒原上建了星汉阁。他最初的尝试全部失败了——小型道痕的稳定性远不如巨型船印——直到他忽然想通:为什么非要稳定？为什么非要一个道痕完成全部任务？

答案是:不要一个，要一百个。不要固定航线，用自动连接。一个失效了，其他九十九个继续工作。这就是星汉阁的核心产业:**星痕浮标网络**。数以千计的小型道痕浮标——每颗只有磨盘大小——悬浮在苍渊高空，彼此之间以道痕共振互相连接，构成一张覆盖整个西部的"空中网络"。修士通过便携式"牵引盘"接入最近的浮标，被网络"接力"传送——换乘的次数越多路程越远但始终在动。不需要大型飞舟。不需要随船技师。不需要维护铁氏家族的专有道痕。

星汉阁的浮标网络已经覆盖了苍渊西部的全部领土。它的覆盖率远不如砥砺宗的航线(浮标毕竟只是浮标，承载不了大宗货运)，但个人修士和小件货物可以以砥砺宗十分之一的价格、三倍的速度从西境的任何地点到达另一端。

### 对比

| | 砥砺宗·铁脊航线 | 星汉阁·星痕网络 |
|---|---|---|
| 核心技术 | 巨型旗舰道痕刻印（《砥砺船术》） | 小型浮标道痕共振（《星衍经》） |
| 优势 | 安全、大宗货运、稳定 | 速度、价格、覆盖灵活、无需技师 |
| 劣势 | 慢、需铁氏技师、航线固定 | 无法运大件、网络有盲区 |
| 收费对象 | 宗门、大宗商队 | 个人修士、中小商户 |
| 历史 | 三百年的垄断地位，铁氏命脉 | 三十年前被砥砺宗赶走的弟子所创 |
| 口号 | "铁脊之上，百里如一" | "你所在处，便是星辰" |

### 功法适配

- **《砥砺船术》**:在船体上刻印巨型道痕，每印一个道痕需要至少三位修炼《砥砺心经》的长老同步协作。单点失效则需铁氏嫡系维修。此法本质上是把飞船当作一个"修炼中的巨人"——人能用砥砺心经淬炼自己，船也能。但船不会自己修炼，必须用铁氏的特殊功法"喂养"道痕。
- **《星衍经》**:将道痕刻印小型化、模块化。单颗浮标的道痕力量极弱——甚至不如人阶初痕修士的随手一击——但浮标之间通过陆川独创的"星图算法"互相校准，以数量补质量。陆川用了十年时间优化星图算法，最后将成功的版本刻在了星汉阁大殿天花板上的每一块砖里——他的继承人只要抬头就能看到。

## 三、势力关联设计原则（后续核心规范）

### 总则

**任何新设计的宗门或势力，必须同时满足以下三条中的至少两条:**

1. **有渊源**——与至少一个已有势力共享历史(同源分裂、师徒关系、旧日恩怨)
2. **有交易**——与至少一个已有势力存在经济/资源上的交换或依赖
3. **有人物**——至少有一个人物与已有势力的人物有私人关系(旧识、同窗、情敌、欠债)

### 核心关系线

1. **沈怀远—铁骊—陆川**  
   沈怀远和铁骊年轻时在青石学宫同窗。铁骊继承砥砺宗后，沈怀远把儿子沈安送到她手上。铁骊当年亲手签了陆川的辞退单——她至今不知道陆川在外面建了一个比自己航线更快的网络。陆川对铁骊没有恨——有某种奇怪的感谢:"如果她没赶我走，我永远不会去荒原。"

2. **云隐宗—砥砺宗**  
   同源分岐。砥砺宗创始人曾是云隐宗长老。两宗关系:看得见的礼貌，看不见的较劲。云隐宗的改良《云隐九诀》使用了叛出长老留下的手稿——这事砥砺宗知道但没人想提。

3. **玉华阁—镇域司—所有人**  
   玉华阁通过苏锦年的隐形保护伞+对镇域司的"友好关系"成为苍渊最不该惹但最欠揍的宗门。镇域司的偏袒来自"上面的压力"——司正不知道是谁建了玉华阁，但他知道有人给玉华阁撑腰。他选择不深究。

4. **青石学宫—沈氏—云隐宗**  
   学宫在沈氏捐赠的旧址上，裘教习的云隐宗旧同窗是爆料出口。周宫主和沈怀远偶有寒暄但从不深谈——沈怀远隐约知道学宫在做什么，但他自己的儿子都管不了，没资格管别人的学生。

### 后续设计检查清单

设计任何新势力时，回答以下问题:
- 谁建的？为什么要建？
- 与哪个已有势力有历史关联？
- 谁恨它？谁需要它？
- 至少有一个核心人物与已有势力的关键人物之间有私人关系。
- 它靠什么活（经济来源/资源交换）？
