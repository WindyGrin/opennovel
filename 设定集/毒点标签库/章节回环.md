# 章节回环

- 类别: 情节
- 触发条件: 章节的起点状态与终点状态完全一致——章末结论等于章首已有结论，整章纹理无信息增量；或前后章结尾句语义重复
- 毒度基准: 6
- 档位锚点:
  | 轻(4) | 结论重复但中间有新纹理，读者不会觉得白读 |
  | 中(6) | 章末=章首，决策无推进，读者可以跳过本章 |
  | 重(8) | 前后章结尾句语义重复，上一章已抵达的决策点在本章被重新抵达 |
- 首次记录: 第8章毒舌审查
- 最近出现: 第8章
- 惯性计数: 0

## 典型案例

> 原文: Ch7末行——"沈怀远当夜做了一个决定。送沈安进砥砺宗。" Ch8末行——"送走。必须送走。"
> 批评: 读者翻到Ch8时决策已经做好。六十七行政治博弈读完，决策还是那个决策。中间多了纹理，没多信息。这章给了一个理由让人跳过。
