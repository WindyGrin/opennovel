# 钩子断裂

- 类别: 情节
- 触发条件: 上一章结尾明确承诺"明天/下一步要做什么"，下一章完全没兑现也没交代
- 毒度基准: 7
- 档位锚点:
  | 轻(5) | 钩子延迟1章兑现，正文中有交代中断原因 |
  | 中(7) | 钩子完全未兑现，无任何文字交代 |
  | 重(9) | 连续2章+钩子断裂，读者对作者的承诺链失去信任 |
- 首次记录: 第22章毒舌审查
- 最近出现: 第22章
- 惯性计数: 0

## 典型案例

> 原文: Ch21结尾"明天他会查筛子的事" → Ch22开头被铁骊传话打断，但没有任何文字说他"被中断了"
> 批评: 你给了读者一个承诺，然后忘了。读者在等，你在写别的。这种断裂会让读者觉得你不在乎他们的时间。
