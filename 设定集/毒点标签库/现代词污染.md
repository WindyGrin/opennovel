# 现代词污染

- 类别: 文笔
- 触发条件: 修仙/古代背景下，正文或角色记录中出现现代科研/管理/技术词汇——"数据""系统""课题""精度""研究""观测""技术路径""物理传导""评估""参考系"等；或分析型角色的思维直接套用现代学术方法论
- 毒度基准: 7
- 档位锚点:
  | 轻(5) | 1-2处边缘词汇，不影响整体语感 |
  | 中(7) | 多处现代词汇或角色思维方式现代化 |
  | 重(9) | 核心段落用现代学术方法论框架，角色像"现代人在cosplay" |
- 首次记录: 第5章
- 最近出现: 第21章
- 惯性计数: 0

## 典型案例

> 原文（Ch5沈青萝竹简）: "研究进入第三日。初步结论：观测对象的高阶感知比我预估的至少高两层。研究对象意识到被观察并主动利用观察者进行自我校验。课题价值升级。需要更多数据。"
> 批评: 聪明人在古代会怎么思考？他们用汉字、用易经、用兵书逻辑、用匠人口诀。沈青萝可以用"察""探""推敲""日课""疑点"——不需要把科研论文搬进竹简里。
