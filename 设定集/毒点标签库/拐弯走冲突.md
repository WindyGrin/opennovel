# 拐弯走冲突

- 类别: 情节
- 触发条件: 章节开头建立明确的对抗性场景（围堵/质问/逼出手），但冲突在发生前被主角用非对抗方式蒸发——提问者忘记来意、围堵者自行散开、对抗消融为零代价
- 毒度基准: 8
- 档位锚点:
  | 轻(6) | 边角料冲突蒸发，不影响章节主冲突线 |
  | 中(8) | 章节建立的核心对抗被非对抗方式消解 |
  | 重(10) | 整章冲突原地消失，读者被架在起跑线上，发令枪是泥捏的 |
- 首次记录: 第7章毒舌审查
- 最近出现: 第23章
- 惯性计数: 1

## 典型案例

> 原文（Ch23）: 三师兄修炼舍外堵路→赵砚清了三次嗓子→问了一句"宗主说了什么"→沈安蹲着看蚂蚁说"等一下"→站起说"后山有人来过"→拍灰走人→赵砚站在原地"信了"
> 批评: 你把冲突架起来了然后自己跑路。三个师兄从"施压者"变成"观众"变成"信众"，中间缺了一整个"为什么"。这不是沈安用智慧化解冲突——冲突根本没发生。
