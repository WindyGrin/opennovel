# 章节截肢

- 类别: 结构
- 触发条件: 章节在核心场景的开头处（而非结束处）切断，核心动作留在大纲中未实现，结尾句是下一幕的开场而非本章的收束
- 毒度基准: 8
- 档位锚点:
  | 轻(6) | 核心场景尾部欠一段收束，主体已完成 |
  | 中(8) | 核心场景在开头处切断，结尾句是开场锣 |
  | 重(10) | 核心动作全空，章末句是下一幕的开场，章节在锣响时断电 |
- 首次记录: 第9章毒舌审查
- 最近出现: 第9章
- 惯性计数: 0

## 典型案例

> 原文: Ch9末行——"第二天清晨。他站在沈氏大门口。" 大纲该章核心爽点（门口对话、说不出口的那句话、沈安转身走）全空。未行是开场锣，章节在锣响时断电。
> 批评: 这不是留白，不是悬念钩。这是写到一半停电了。读者翻页以为好戏开演，看到的是下一章标题。
