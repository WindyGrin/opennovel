# 反派工具化

- 类别: 角色
- 触发条件: 反派动机只有一个维度——"想逐出主角"；反派台词像公告而非人话；反派的存在意义仅在于与主角对位
- 毒度基准: 8
- 档位锚点:
  | 轻(6) | 反派动机略单一但台词自然，有独立于主角的行为片段 |
  | 中(8) | 单向动机+台词公告化，恶不扩散=只为剧情服务 |
  | 重(10) | 反派存在意义仅在于与主角对位——没有自己的目标/恐惧/日常 |
- 首次记录: Ch7/Ch27/Ch31原始设计
- 最近出现: Ch8
- 惯性计数: 0

## 典型案例

> 原文（单向动机）: 沈崇武在Ch7中除了"写报社论影射沈安"外无任何独立行为——他对底层子弟的真·残酷（批量逐出）和对沈安的虚·恐吓（再议再议）之间没有建立讽刺连接点
> 批评: 最好的反派不知道自己正在当反派。沈崇武如果在逐出底层子弟时效率极高、在对付沈安时反复"再议"——这个温差本身就是喜剧。反派应该是一个体系的一部分，而不是针对主角的工具人。
