# 匠气太重

- 类别: 文笔
- 触发条件: 全书已建立的概念（如"磨""刚好"）反复加引号强调；或特定标点符号（如"——"）密度过高成为句法拐杖；或同一角色特征词在同章反复出现≥3次
- 毒度基准: 6
- 档位锚点:
  | 轻(4) | 1-2处复用，不形成感知模式 |
  | 中(6) | 3-5处复用，标点/特征词开始可预测 |
  | 重(8) | 6处+复用，或多类型并发（标点+特征词+引号同时出现） |
- 首次记录: 第23章毒舌审查
- 最近出现: 第5章
- 惯性计数: 0

## 典型案例

> 原文（Ch5）: 93行正文出现15+处"——"破折号，平均6行一根；"幅度极小"x3、"眉毛没动/动了/波动"x3、眼睛半合x3
> 批评: 破折号成了句法拐杖，角色特征词复用成条件反射。不是"风格统一"，是"词汇库见底"。
