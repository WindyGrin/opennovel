# 废材标签堆砌

- 类别: 角色
- 触发条件: 用"废物三年""公认的废物""全族都看不起他"等标签反复贴牌代替行为描写
- 毒度基准: 6
- 档位锚点:
  | 轻(4) | 1-2处标签式叙述，仍有场景支撑 |
  | 中(6) | 多处标签代替行为描写，读者通过标签认识角色而非场景 |
  | 重(8) | 角色认知全部来自标签堆砌，零场景支撑——"全族看不起"从未被演出来 |
- 首次记录: 简介/人物卡修正
- 最近出现: 第1章
- 惯性计数: 0

## 典型案例

> 原文: 简介/初期人设中反复出现"废物""躺平二代""三年没动过"等标签
> 批评: 标签是给AI看的提示词，不是给读者看的故事。你写他被质疑时点头了——读者自己会得出结论。偷懒写标签不仅废文笔，还废了读者自己做判断的乐趣。
