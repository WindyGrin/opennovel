# 动机链缺失

- 类别: 角色
- 触发条件: 主角答应帮人/教人但未解释"为什么这个怕麻烦的人会破例"；或主角重大行动的触发点是"亲人受辱"等被动廉价情感而非角色自身性格缺陷驱动
- 毒度基准: 7
- 档位锚点:
  | 轻(5) | 动机可推断但正文未交代，不影响理解 |
  | 中(7) | 关键行动缺动机链一环，读者会问"他为什么这么做？" |
  | 重(9) | 主角行动违背核心人设——怕麻烦的人主动惹麻烦，无成本核算 |
- 首次记录: Ch20-22/Ch36-40原始设计
- 最近出现: Ch58
- 惯性计数: 0

## 典型案例

> 原文（主动帮人缺动机）: 沈安答应教沈小禾道痕——为什么？大纲说"对照实验"——但正文里读者看到的是"忽然答应了"。中间缺一句"不教的话沈守拙会天天来找→教了反而每周只需一次→省时间。他算过。"
> 批评: 沈安的每一个"多管闲事"都必须经过精确的成本核算。怕麻烦的人只有在"不插手会更麻烦"时才会出手。如果读者读完之后感觉"他就是个好人嘛"——你写错了沈安。
