# 大纲脱节

- 类别: 结构
- 触发条件: 章节实际内容与大纲条目显著偏离。**⚠️ 关键区分**：偏离≠错误。判断方向是——章节本身好不好？
- **章节好 → 改大纲**：章节有独立生命力，精彩、有新意、读者会追读。大纲是它诞生前的草稿，草稿应该让步于成品。
- **章节差 → 改章节**：章节本身虚弱（节奏断裂/情绪空转/核心爽点丢失），大纲的方向比当前的执行更健康。
- 毒度基准: ※ 不定。章节差且偏离大纲 → 8；章节好且偏离大纲 → 不评定为崩点，转而标注"大纲需更新"
- 档位锚点（仅章节差时适用）:
  | 轻(6) | 小幅偏离，核心场景仍在，张力略降 |
  | 中(8) | 核心场景被替换，冲突张力明显弱于大纲设计 |
  | 重(10) | 整章漂移，大纲骨架错位，核心动作全空 |
- 首次记录: 第5章毒舌审查
- 最近出现: 第8章
- 惯性计数: 0

## 典型案例

> ⚠️ 章节差 → 改章节（毒度 8）: Ch7大纲要求"沈崇武借族报发难、两人矛盾浮上台面"，实际全章沈崇武零出场，冲突被替换为案头读文件。
> ✅ 章节好 → 改大纲（不评定为崩点）: 如果某章偏离了大纲的走向，但读起来紧凑有力、角色亮眼、情感到位——那就不是章节的问题。
