# 句式复读机

- 类别: 文笔
- 触发条件: 同一章节内相同句式（如"不是X而是Y"）出现≥5次
- 毒度基准: 7
- 档位锚点:
  | 轻(5) | 5-7次，出现在非关键段落，不打断核心场景节奏 |
  | 中(7) | 8-12次，开始形成可感知的模式，影响阅读流畅度 |
  | 重(9) | 13次+，或核心场景中出现密集复用，读者产生烦躁感 |
- 首次记录: 第23章毒舌审查
- 最近出现: 第23章
- 惯性计数: 1

## 典型案例

> 原文: "赵砚脑子里第一个反应不是愤怒——是困惑。""不是蚂蚁懂得道痕理论——是它们的身体比重..."
> 批评: 全章20处"不是X是Y"，读到第5处开始加速翻页。这不是风格，是写作肌肉记忆的塌方。
