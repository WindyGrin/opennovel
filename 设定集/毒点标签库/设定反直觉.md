# 设定反直觉

- 类别: 设定
- 触发条件: 世界观细节设计虽然在体系内自洽，但违背读者的文化直觉或常理——读者第一反应是"这不对吧"而不是"这个设定有意思"
- 毒度基准: 8
- 档位锚点:
  | 轻(6) | 细节反直觉但读者可快速接受，不影响沉浸 |
  | 中(8) | 读者第一反应是困惑而非沉浸，需要额外解释 |
  | 重(10) | 严重违背文化常识，破坏世界观的"真实感" |
- 首次记录: 第1章(初版)
- 最近出现: 第1章
- 惯性计数: 0

## 典型案例

> 原文（Ch1初版）: "石碑上刻满了被逐出者的名字"——中国宗族传统中石碑刻名是表彰/纪念性质，刻逐出者的名字逻辑不自洽
> 批评: 一个设定即使在体系内严丝合缝，如果读者的第一反应是困惑而非沉浸——那就是设计失败。修复方案：改为"族谱朱笔勾名"——读者不需要任何解释就能接受。
