# 模式耗尽

- 类别: 情节
- 触发条件: 同一情感节拍/角色行为模式（如"咽话"）在连续章节中机械复用≥4次，情绪电压递减至零，每次复用未增加任何新维度
- 毒度基准: 7
- 档位锚点:
  | 轻(5) | 第3-4次复用，已现疲态但尚未归零 |
  | 中(7) | 第5-6次复用，情绪电压归零，读者开始烦躁 |
  | 重(9) | 7次+复用，最关键的一次引爆点也用完全相同的句式执行——连换件衣服都不肯 |
- 首次记录: 第10章毒舌审查
- 最近出现: 第10章
- 惯性计数: 0

## 典型案例

> 原文: Ch2廊下咽话、Ch2B槐树前咽话、Ch3族谱阁咽话、Ch8牌位前咽话、Ch9廊道拐角咽话、Ch10大门口咽话——六章内同一"脑内排一遍→找不到嘴→咽了→换成无关的话→双方都知道没说"的模板
> 批评: 第一次咽话是深情。第三次是惯性。第六次是他妈的词穷。沉默不是唯一的深度，一本书写六次同款沉默是作者手里只剩这一张牌。
