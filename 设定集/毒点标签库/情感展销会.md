# 情感展销会

- 类别: 结构
- 触发条件: 单章内情感锚点（回忆物品/场景/对话）≥4个，每个平均篇幅不足15行，读者来不及沉浸即被切换
- 毒度基准: 7
- 档位锚点:
  | 轻(5) | 3-4个锚点，切换较快但各自相对完整 |
  | 中(7) | 4-5个锚点，平均<15行，读者情绪被频繁切换 |
  | 重(9) | 5个+锚点，碎片化展览，每个单独抽出来都是好段落，叠在一起变成PPT导览 |
- 首次记录: 第9章毒舌审查
- 最近出现: 第9章
- 惯性计数: 0

## 典型案例

> 原文: Ch9——43行正文塞入旧信、描红、青石、牌位、廊道五个情感锚点，平均每8.6行切换一次，核心门口场景零行
> 批评: 每一个单独抽出来都是好段落，叠在一起变成PPT导览。读者刚为青石起了鸡皮疙瘩，下一秒已被拖到牌位前——鸡皮疙瘩在上一个展厅原地冷却了。
