# 基调偏移

- 类别: 结构
- 触发条件: 全书定位为轻松讽刺文，但章节叙事语调偏沉重/悲凉/严肃，让事实本身说话的比例<60%，叙事者替角色宣泄情绪或渲染压抑氛围
- 毒度基准: 7
- 档位锚点:
  | 轻(5) | 少数段落偏严肃，主体仍在讽刺框架内 |
  | 中(7) | 叙事语调整体偏沉重，讽刺锚点不足 |
  | 重(9) | 全章丧失讽刺感，让事实说话<40%，叙事者替读者感受 |
- 首次记录: 第1章(初版)
- 最近出现: 第1章
- 惯性计数: 0

## 典型案例

> 原文（Ch1初版）: "碑面是凉的。秋后清晨的石碑，碑身从地脉里吸了一夜的寒气……""他什么都知道。他一个字都不说。"——叙事者替老石匠解释了他的沉默，把"让事实说话"变成了"替你感受"
> 批评: 全书公式是"认真残酷的世界+不认真的主角=温差=笑"。修复方向：让事实自己产生荒谬感，不让旁白替读者感受。
