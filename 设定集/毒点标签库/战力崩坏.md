# 战力崩坏

- 类别: 设定
- 触发条件: 力量体系本身边界模糊——每境的力量呈现/战斗表现/跨境压制逻辑未定义；或道痕可见性/感知规则在正文中表述混乱；或主角力量来源表述矛盾
- 毒度基准: 9
- 档位锚点:
  | 轻(7) | 单个边缘能力表述模糊，不影响核心战斗逻辑 |
  | 中(9) | 关键能力来源与设定矛盾，但可通过修正表述解决 |
  | 重(10) | 整个战力体系边界不清，核心战斗逻辑无法成立，写作时无画面可参照 |
- 首次记录: 第5章
- 最近出现: 第14章
- 惯性计数: 0

## 典型案例

> 原文（来源混乱）: 沈安感知天痕之脉的行为被描述为窥痕境的能力，但按照设定窥痕境·初痕连感知天痕之脉的资格都没有——他能感知是因为执痕残留，不是窥痕境界的功劳
> 批评: 战力崩坏不一定是"他昨天打不过今天打赢了"——更隐蔽的是"体系没画清楚"。沈安的每一次"超纲感知"都必须让读者意识到是封印裂痕的代价——不是窥痕境·初痕自带的福利。
