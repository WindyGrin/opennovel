# 节奏掉崖

- 类别: 情节
- 触发条件: 动作/冲突段落突然切到纯内心独白或环境描写，时间流动被截断
- 毒度基准: 5
- 档位锚点:
  | 轻(3) | 短内心独白穿插（<5行），张力基本保持 |
  | 中(5) | 中段内心戏打断动作流，读者感到节奏被截断 |
  | 重(7) | 长段落独白冲散核心张力，前面建立的紧张感被冲干净 |
- 首次记录: 第23章毒舌审查
- 最近出现: 第23章
- 惯性计数: 1

## 典型案例

> 原文: 蚂蚁拐弯（冲突）→ 沈安站起来 → 铁骊偏殿独白（内心戏长段落）
> 批评: 前面刚建立起来的张力，被你用一段铁骊独白冲得干干净净。你怕读者看不懂吗？读者不需要你帮他们消化每件事。
