# 旁白抢戏

- 类别: 叙事
- 触发条件: 角色行为发生后旁白立刻跳出解释"为什么别人做不到"；或第二人称"你"闯入POV替读者下判断；或旁白以全知视角完整转述角色内心独白附带因果链分析
- 毒度基准: 7
- 档位锚点:
  | 轻(5) | 1-2处轻量解释，不破坏留白和沉默的张力 |
  | 中(7) | 多处旁白跳出来"翻译"角色行为，读者失去自行解读空间 |
  | 重(9) | 解释性旁白占比>30%，角色的沉默/留白全部被翻译为公开信息 |
- 首次记录: 第22章毒舌审查
- 最近出现: 第8章
- 惯性计数: 0

## 典型案例

> 原文（Ch8）: 全章65行，解释性旁白占比超60%。每行角色动作后面都站着一个"解说员"用3-5行替读者解码——端茶解释、走路解释、眉毛解释、敲桌解释、印子深浅解释。
> 批评: 本该读者自己咂摸的余味全被嚼烂了喂到嘴边。两个高手的沉默对弈本来够锋利，被你用解说词泡软了。
