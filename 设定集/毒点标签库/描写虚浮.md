# 描写虚浮

- 类别: 文笔
- 触发条件: 景物/天气描写无叙事功能——"风大""天凉""晨风冷"等空壳套话反复出现；或景物段落情绪不匹配场景氛围；或以自然意象替代情绪本身但无具体叙事支点
- 毒度基准: 6
- 档位锚点:
  | 轻(4) | 2-3处空洞景物，不影响叙事推进 |
  | 中(6) | 多处景物段落无叙事功能，纯为填充 |
  | 重(8) | 高频复用同质景物词语（如全章7处风描写），形成风格疲态 |
- 首次记录: 第1章(初版)
- 最近出现: 第1章
- 惯性计数: 0

## 典型案例

> 原文（Ch1初版）: "碑前风大""今晨的风比昨天冷了半度""风里面那个说不清的东西"——全章7处风描写，无一处有叙事功能
> 批评: 借景抒情是对的——但"风大"不是抒情，是偷懒。风不是万金油——在没有飓风危机、没有御风战斗的段里反复写风，就是在给读者的白眼煽风。
