# 大纲正文偏离

- 类别: 一致性
- 触发条件: 章节正文实际内容与卷纲详细大纲条目显著偏离。**⚠️ 关键区分**：偏离≠错误。判断方向是——大纲条目规划得好不好？正文写得好不好？
- **正文好 → 改大纲**：正文有独立生命力、精彩有新意。大纲是它诞生前的草稿，草稿应该让步于成品。
- **正文差 → 改正文**：正文本身虚弱（节奏断裂/情绪空转/核心爽点丢失），大纲规划的方向比当前执行更健康。
- 毒度基准: ※ 不定。正文差且偏离大纲 → 8；正文好且偏离大纲 → 不评定为崩点，转而标注"大纲需更新"
- 档位锚点（仅正文差时适用）:
  | 轻(6) | 小幅偏离，核心场景仍在，张力略降 |
  | 中(8) | 核心场景被替换，冲突张力弱于大纲设计 |
  | 重(10) | 整章漂移，大纲骨架错位 |
