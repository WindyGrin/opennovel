# 伏笔孤儿

- 类别: 逻辑
- 触发条件: 总纲伏笔表的"埋设章"或"回收章"在对应卷纲的详细大纲中不存在对应条目。包括孤儿伏笔（只埋不收——有埋设大纲条目但回收章所在卷纲条目缺失）和幽灵伏笔（只收不埋——有回收大纲条目但埋设章所在卷纲条目缺失）
- 毒度基准: 9
- 档位锚点:
  | 轻(7) | 1条伏笔的回收章号偏移≤3章，大纲未同步更新 |
  | 中(9) | 2-3条伏笔无对应大纲条目 |
  | 重(11→钳制10) | ≥4条伏笔孤儿，或核心伏笔（标记为"核心"的伏笔）成为孤儿 |

- 最近出现: -
- 惯性计数: 0
