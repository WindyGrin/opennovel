## 关系速查

```
沈安 ←(父亲)→ 沈怀远（打不过/沉默送走了）
  │
  ├←(互相研究)→ 沈青萝（轨迹图/交换笔记）
  ├←(沉默同盟)→ 温小鹿（不说话=最舒服状态）
  ├←(送鱼)→ 云渡（沈安：能躲就躲）
  │
  ├←(欠两顿酒)→ 沈守拙（"我知道是你"）
  ├←(被迫营业)→ 厉淮（暗中放水）
  ├←(黑历史交易)→ 鉴老
  ├←(灌醉认爹)→ 拭痕（"爹你还欠我一坛酒"）
  │
  ├←(前世兄弟)→ 萧长河（"那坛酒还在吗"）
  ├←(年年来喝茶)→ 谷焚天（"明年我还来"）
  ├←(进制多一位)→ 岑不言（岩壁破例）
  └←(终极对手)→ 苏锦年（怕死→救世→疯狂）
```

## 升级规则

1. 次要角色如需跨卷持续出现（≥2卷）→ 升级为主要角色，补充深度
2. 次要角色如转为沈安的阻力来源 → 升级为反派角色
3. 次要角色如任务完成（如裘教习的揭发任务）→ 保持次要，不再升级
4. 新角色入场 → 默认放入次要角色，一卷后评估是否升级
