# 力量体系设定

## 双体系概述

本项目中的修仙世界存在**两套并行的修炼体系**：

| | 传统体系（练气筑基） | 道痕体系 |
|---|---|---|
| **适用区域** | 玄霄、虚海 | 苍渊 |
| **核心依赖** | 灵气（天地能量） | 悟道（对法则的理解） |
| **根基** | 旧纪元法则 | 独立于法则的"刻印" |
| **纪元更迭影响** | 法则崩塌 → 修为清零 | 不受法则替换影响 |
| **创造者** | 远古传承 | 沈溪风（渡劫前约三百年创造） |

## 为什么有两套体系

沈溪风前世是玄霄最顶级的道痕大师——别人修炼是吸灵气、练功法，他修炼是研究"法则本身到底是什么"。他逐渐发现一个所有人都不愿意面对的事实：玄霄的修炼体系建立在旧纪元法则之上。而纪元更迭一旦发生，旧法则被替换，所有基于旧法则的修为都将化为乌有。

他把这个推测告诉了少数人——没人信。有人觉得他在找借口偷懒（他确实经常偷懒），有人认为他杞人忧天。于是他做了一件只有他做得出来的事：不再说服任何人。他自己动手，创造了一套不依赖任何特定法则的修炼体系——道痕体系。

道痕体系的核心思路是：修为不等于你储存了多少天地能量，而是你在自己的神魂上刻下了多深的理解。法则可以替换，但你刻在自己身上的东西不会被抹掉。就像一本书——纸张可以换，但刻在你脑子里的内容不会消失。

他把这套体系的"种子"引入了苍渊。

苍渊是天然的实验场——这里灵气最稀薄，法则束缚最弱，传统修炼体系在这边几乎不成立。但也正因为如此，这里是测试"不依赖法则的修炼"最理想的地方。他一开始只是埋了一个幼苗——一种新的灵气运转逻辑、一套新的突破方式。然后他就没有再管它。苍渊修炼者自己发现、自己摸索、自己丰富了这套体系。他们不知道这套东西有个名字叫"道痕体系"，更不知道它的创始人是一个在玄霄晒太阳的懒鬼。

而玄霄至今仍在使用传统体系。沈溪风没推广道痕体系的理由很简单——推广起来太麻烦。他不想开宗立派，不想授课传道，更不想应付那些"你怎么证明纪元会崩塌"的问题。他把种子种下去了。种子的后续生长与他无关。

这个逻辑同时解释了苏锦年的选择——她所知的"法则保险"根本不存在，因为沈溪风创造的道痕体系不是一件可以抢走的"东西"。它是一种思想。思想杀不死，也偷不走。

## 道痕体系详解

### 境界链

```
人阶    地阶    天阶
初痕    刻痕    道痕
  ↓       ↓       ↓
明痕 → 深痕 → 融痕 → 破痕 → 化痕 → 归痕
```

每境分初/中/后/圆满四小层。跨越大境界比传统体系更难——因为道痕突破需要的是"开悟"而非"积累能量"。

### 核心公理

- **突破靠悟道**：不理解法则的人，无论如何吸收灵气都无法突破。理解到了，自然就到了。
- **与灵气无关**：道痕修炼者仍然可以吸收灵气——灵气是工具，不是燃料。就像一把刀——不需要它自己产生能量，但用好了可以切很多东西。
- **越逃避越容易悟道（沈溪风的特殊悖论）**：道痕的本质是"旁观法则"。当你处于旁观者的位置，不参与、不强求、不执迷时，你的心智最澄澈，对法则的感知最精确。沈溪风的躺平恰好创造了这种最佳的旁观状态。他不是"躺平变强"——他是"因为躺平所以看得更清楚，因为看得更清楚所以突破更快"。
- **公平性**：道痕体系不依靠血脉、天赋、资源堆叠。任何人只要能"理解"，就能突破。但它也有一个残酷的公平：有些法则，你的人生阅历不够深，就永远理解不了——跟聪明无关，跟你经历了什么有关。

### 与传统体系的差异

| 维度 | 传统体系（练气筑基） | 道痕体系 |
|---|---|---|
| 突破方式 | 积累灵气 → 冲破桎梏 | 理解法则 → 自主刻印 |
| 修炼速度 | 靠天赋、丹药、机缘 | 靠悟性、阅历、心态 |
| 寿命 | 每突破一大境大幅延长 | 延长，但不显著——因为力量不依赖于生命能量 |
| 越级战斗 | 有硬性灵气差距，很难越级 | 可以越级——如果对方不理解你理解的法则 |
| 纪元后存活 | ❌ 修为大概率归零 | ✅ 刻印在神魂上，无法抹除 |

### 苍渊道痕体系的演化

道痕体系在苍渊落地后，被苍渊的修炼者自己发展出了完整的文化和结构：

- 苍渊的"宗门"不完全依赖资源垄断——更依赖"传承"（一个前辈把他对某条法则的理解传下来，后辈在此基础上继续深入）。
- 苍渊有"道痕试炼"——一种公开的、靠意志和悟性而非武力的竞争方式（类似科举，但考的是对法则的理解）。
- 道痕体系的最高境界目前是"天阶·归痕"——苍渊至今无人到达。大部分修炼者终生在人阶和地阶之间。天阶被认为只存在于理论中——直到沈溪风下凡。

## 关键人物与道痕体系

- 沈溪风（创造者）：前世用拭痕刻下了道痕体系的第一道种子。这一世他需要在苍渊重新理解自己创造的体系——因为苍渊已经把它发展成了一种他都不完全熟悉的新形态。
- 萧长河（不知情者）：一直在用传统体系。如果他再坚持传统体系直到纪元更迭，他将面临修为归零。苏锦年不知道自己在无意中让他成了"两个体系都接触过"的极少数人之一。
- 苏锦年（误会的核心）：她以为是沈溪风藏起了某种"法则保险"。她不知道"法则保险"不在任何一件宝物上——它在苍渊。那个被她赶下凡的人，把一份无形的遗产留在了所有人脚下。她花了一生找的东西，自己正踩在上头。
