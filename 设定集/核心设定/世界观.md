# 世界观设定

## 界域结构

本故事的世界由三层界域构成，从下到上依次为：

### 苍渊（人间界）
- **特征**：荒古大陆，灵气浓度极低。法则束缚最弱——正因如此，这里是道痕体系生根发芽的土壤。
- **修炼体系**：道痕体系（窥痕境→刻痕境→化痕境），由沈溪风约三百年前用拭痕刻下第一道"种子"投入苍渊。修炼靠"悟道"刻印天地法则于己身，与灵气无关。
- **势力格局**：苍渊的宗派、世家、学宫与地下之络交织成一张松散的网。老牌宗派如云隐宗隐于云雾山脉数百年不动；新兴宗派如砥砺宗以铁脊航线崛起于近百年；星汉阁更是在三十年内靠周天星痕异军突起。世家层面——青石城里的沈氏与焚金谷谷氏隔着一座矿山共存、共耗、共用一条矿脉。清河崔氏以辨痕之术世代填充着镇域司的调查处。学宫层面上——青石学宫来者不拒却已濒临关门；叩鸣堂从不缺学生但永远"今年座满"。地下——影阁贩售情报，夜鸮承接暗杀，蜕庐居中交易。玉华阁作为苏锦年留在苍渊的暗棋，以全女宗门的面目运作着牵痕术。镇域司在所有这些力量之间端着一碗永远端不平的水。
- **交通**：砥砺宗铁脊航线(大型飞舟，慢但稳) vs 星汉阁周天星痕(浮标接力，快但只能运小件)。隔阂的壁垒。

### 虚海（中间界）
- **特征**：界域之间的过渡地带，没有连续大地——由无数浮岛和破碎空间构成，法则在此处扭曲混乱，正常修炼难以进行。
- **修炼体系**：传统体系为主(从玄霄传承)，道痕体系刚刚开始传入。虚海本身不产出修炼者——它的居民是从苍渊或玄霄迁入的各类"边缘人"。
- **势力格局**：渡舟人(穿梭界域的交通组织，掌控《虚舟诀》)、墟市(三界唯一自由贸易区，以信息垄断控制交易)
- **交通**：渡舟人的界域渡船——在法则扭曲的虚海中找到裂缝穿越。

### 玄霄（天界）
- **特征**：法则最密集的界域，旧纪元法则的中心。仙门林立，万古势力盘踞。纪元更迭正从此处蔓延——旧法则在崩塌，新法则在生长。这是苏锦年恐惧的来源。
- **修炼体系**：传统体系(练气→筑基→金丹→元婴→化神→合体→大乘→渡劫)。所有修为依赖于旧纪元法则——纪元更迭后大气运清零。
- **势力格局**：萧氏丹阁(炼丹世家·萧长河)、苏氏(联姻世家·苏锦年母族)、天律司(仙界执法机构·厉淮)、纪元观测者(隐藏势力·纪元研究)
- **交通**：已故老艄公："四百年前也有个人从玄霄来。他走的时候没付船钱。"

## 修炼体系双轨制

| | 传统体系(练气筑基) | 道痕体系 |
|---|---|---|
| 使用区域 | 玄霄、虚海 | 苍渊 |
| 核心依赖 | 灵气 | 悟道(对法则的理解) |
| 根基 | 旧纪元法则 | 刻印在神魂上，法则无关 |
| 纪元影响 | 崩塌→清零 | 常驻，不受影响 |
| 创造者 | 远古传承 | 沈溪风(约300年前) |
| 突破脉理 | 积累→冲破桎梏 | 理解→自主刻印 |
| 寿命 | 突破大幅延长 | 延长但不显著 |
| 越级战斗 | 有硬性差距，难 | 可以越级(理解差) |

## 道痕体系核心

- **突破靠悟道**：不理解法则的人，无论如何吸收灵气都无法突破。
- **与灵气无关**：道痕修炼者仍可吸收灵气——灵气是工具不是燃料。
- **越逃避越容易悟道（沈安悖论）**：道痕的本质是"旁观法则"。处于旁观者位置时心智最澄澈，感知最精确。沈安的躺平恰好创造了最佳旁观状态。
- **公平性与残酷性**：不依靠血脉、天赋、资源堆叠。但有些法则你需要经历足够深才能理解——跟聪明无关。

## 道痕境界链

```
窥痕境  刻痕境  化痕境
初痕    深痕    道痕
  ↓       ↓       ↓
明痕 → 深痕 → 融痕 → 破痕 → 化痕 → 归痕
```
每境分初/中/后/圆满四小层。化痕境·归痕在苍渊历史上无人到达——只在理论中存在。

## 道契与牵痕术

> 道痕体系的第一个"人际关系"应用。详见 `设定集/核心设定/道契设定.md`。

**道契**——由萧长河（约三百年前）创造：将双方的约定以"镜痕"的形式刻印在天痕之脉上。本质是**法则层面的契约**，不可磨灭，天道自行执行。

**修为锚定**——萧长河为苏锦年签下第一份道契，将她的修为锚定在天痕之脉上。这是苏锦年能在纪元更迭中保住修为的真正原因——不是她找到了什么"法则保险"，是萧长河在她不知道的时候已经把保险放在了天痕之脉深处。

**牵痕术**——玉华阁基于道契根基发展出的修为分流秘法。利用"自愿"定义的可操纵性，诱导男性修士在不知情的情况下签订契约，修为被持续单向分流。合乎规矩、无人能循迹追查、层层推托无人肯问。

**沈安的视角**：作为能感知天痕之脉底层的前执痕，他能"看到"苍渊有多少份《牵痕术》契约在运行。但他不能全部解除——每解除一份，他在网络上的存在就更清晰一分。救所有人 = 暴露自己。

## 纪元更迭

- **本质**：旧纪元法则被新纪元法则替换。不是毁灭——是更新。但建立在旧法则之上的所有修为将化为乌有。
- **触发者**：未知。纪元观测者是最接近答案的人——但他们从不干涉。
- **影响范围**：从玄霄开始向外蔓延。苍渊距中心最远，受波及时间最晚。
- **苏锦年的恐惧**：她的全部修为都建立在旧法则之上。她不信任道痕体系——她以为"法则保险"在沈溪风手上。她不知道沈溪风已经把保险留在了她脚下。
- **沈溪风的先见**：他早在三百年前就预见了这一切。他从不说服任何人——太麻烦。他自己创造了一套解法，然后把它种在了苍渊的土地里。

## 世界观关键矛盾

1. **道痕 vs 传统**：两种修炼体系的冲突将在纪元更迭时迎来终极对决。
2. **苍渊的独立性**：苍渊是唯一不需要依赖旧法则的界域。这让它同时是"避难所"和"被争夺的棋子"。
3. **所知不等**：苍渊的绝大多数人不知道纪元更迭的存在。知道的人中——苏锦年误解了它、沈安在躲避它、萧长河尚未被告知。
4. **法则与人**：道痕体系的核心命题是——法则可以替换，刻在你神魂上的东西不会消失。这是对纪元更迭的唯一解法，也是最终章沈安与苏锦年的对弈的主题。
5. **大势力与小势力**：苍渊没有统一的权力中心。大宗派之间维持着一种不需要约定的默契——谁也不先动手，因为谁先动手谁先暴露出自己的盟友是谁。但中小势力没有这种奢侈——焚金谷的炉子不能停，青石学宫的门不能关，沈氏的族会不能不开。它们在大势力的夹缝中靠每一天的微小决定活着。沈安恰好出生在其中之一的夹缝正中间。
