# 创作基调约束（全书核心规范）

## 一、文风定位

本书是**轻松搞笑文**。不是"仙侠喜剧"——是"一个很认真很残酷的仙侠世界里，有一个很不认真的主角，把所有人的认真都变成了笑话"。

## 二、禁止项

- ❌ **苦大仇深**：主角没有血海深仇。他前世是自己作死（渡劫假死）、自己选择下界的。今世最大的困扰是"太麻烦"和"兄弟要来找我问话"。
- ❌ **传统退婚/废材/被逐出家族**：不开局卖惨。主角在家族中的地位是"公认的废物"，但不是"被欺负的废物"——他是戒律长老的独子，住好院子吃好丹药，族人看不起他但他不在乎。
- ❌ **套路化装逼打脸**：不打脸。不打脸不等于没冲突——冲突用"反套路"解。

## 三、反套路打法（替代传统打脸）

### 3.1 传统 → 本书替代方案

| 传统套路 | 本书反套路 |
|---|---|
| 对手嘲讽→主角爆发碾压→全场震惊 | 对手嘲讽→主角懒得动手→对手自己失误把自己解决了→全场沉默→主角:"运气。绝对是运气。" |
| 隐藏实力被揭穿→众人倒吸凉气 | 隐藏的实力被"揭穿"了——但揭穿的方式极其社死（比如嘴瓢说了拭痕的梦话），众人觉得这人八成有病 |
| 天才少女看不起主角→主角展现实力→天才少女脸红心动 | 天才少女认真分析了主角的一切行为→得出了"此人必有大秘密"的结论→疯狂研究他→主角吓得以为她要找他麻烦 |
| 美女主动示好→主角感动 | 美女主动示好→主角躲得更远，在心里给"最危险的女性"排名更新了 |

### 3.2 核心原则

- **主角不出手。** 道痕感知让他可以用概率和巧合来解决问题——问题不是他解决的，是世界帮他顺带解决的。
- **观众的反应比主角的动作好笑。** 主角做的事很无聊（打个哈欠、换个姿势、说了一句"麻烦"），但周围所有人的解读极其精彩。
- **爽点来自"别人怎么想这件事"而非"这件事本身"。**

## 四、喜剧规则

1. **主角的嘴贱是最可靠的武器。** 不是骂人——是他在最不应该开玩笑的场合说了一句大实话，所有人都不知道怎么接。
2. **任何人想"教育"他、改变他、激励他——最后都会变成被他"教育"。** 不是靠道理和实力，是靠他自然而然地把对方的认真变成了一件看起来很搞笑的事。
3. **误会是喜剧的第一生产力。** 但误会必须合理——不是"刚好没解释"，是"解释了但对方理解成了完全相反的意思"。
4. **配角越认真越像笑话。** 砥砺宗全员奋斗→主角在旁边眯着眼晒太阳→观众不是在笑主角，是在笑这座"全员鸡血"的宗门。



## 五、势力设计原则（核心规范）

任何新设计的宗门或势力，必须同时满足以下至少两条：

1. **有渊源**——与至少一个已有势力共享历史（同源分裂、师徒关系、旧日恩怨）
2. **有交易**——与至少一个已有势力存在经济/资源上的交换或依赖
3. **有人物**——至少有一个核心人物与已有势力的人物有私人关系（旧识、同窗、情敌、欠债）

禁止孤立设计。新势力不与旧势力产生关联的，视为不合格设计。
