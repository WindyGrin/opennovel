# 力量体系设定

## 双体系概述

| | 传统体系(练气筑基) | 道痕体系 |
|---|---|---|
| 适用区域 | 玄霄、虚海 | 苍渊 |
| 核心依赖 | 灵气 | 从"描红"到"悟道"的渐进理解 |
| 根基 | 旧纪元法则 | 刻印在神魂上，法则无关 |
| 纪元影响 | 崩塌→清零 | 常驻，不受影响 |

## 道痕体系：从描红到执痕

> 道痕体系不是一上来就要"悟"——那是对新手的不公平。它被设计为一条从模仿到创造的渐进之路。

### 窥痕境（描红阶段 · 对应传统练气→筑基）

**突破方式**：跟随师门传承或已有功法中记录的道痕路径，反复描摹、抄录、印入己身。

**不需要理解。** 道痕有"传导性"——前人刻下的痕迹可以被后人临摹，从而在自己身上留下浅层刻印。就像学书法先描红——你不需要懂为什么这一笔要这么写，你只需要把笔顺练熟。窥痕境的全部任务就是"看"和"描"——看见前人留下的道痕，把它们临摹到自己身上。**完全不涉及法则感悟。**

**典型场景**：沈氏家族的《青石经》（残本）就是一套供弟子描摹的道痕路径。云隐宗的入门弟子在最初半年做的就是反复描摹《云隐九诀》的起手式道痕。砥砺宗的底层弟子每天在做的是把宗训上的"天道酬勤"四个字描三百遍——描着描着修为就上去了（然后他们以为是自己够勤奋）。

**子层**：初痕 → 描痕 → 明痕 → 透痕
- 初痕：第一次成功描摹，体表浮现第一道浅痕（沈安表面境界）
- 描痕：能稳定重复描摹多条道痕路径
- 明痕：描摹的道痕开始自发显现，无需刻意调动
- 透痕：描摹的道痕深入肌理，准备好进入下一境

### 刻痕境（自痕阶段 · 对应传统金丹→元婴）

**突破方式**：必须在至少一条法则上形成自己的独立理解。不能再靠描——**必须自己刻。**

**原理**：临摹的痕迹是浮在表面的，可以被更强的人抹掉。只有自己刻下去的道痕——基于自己的独立理解——才会跟自己的神魂融在一起。这一步开始需要"悟"——但这里的"悟"不是感悟天地法则的大彻大悟，只是"我想通了一件事"。

**法则接触**：刻痕境的修炼者**初步接触法则的边界**——不是理解和掌握法则，而是"看到了法则在某个具体问题上的表现"。比如"云为什么不会掉下来"——这不叫理解云雾之道，这叫理解了上升气流与道痕共鸣之气机的一个边界条件。这种级别的"悟"足以推动突破，但离真正掌握法则还差得远。

**举例**：云隐宗的一位长老，他的刻痕境突破来自一次雷雨天在云雾山脉最高峰上站了一整夜——他想通了"云为什么不会掉下来"。不是理解了整个云雾之道——只是理解了一条极小的、关于"上升气流与道痕共鸣之气机"的边界条件。但这一条足够他突破到刻痕境·刻痕。

**沈安的悖论**：他不需要描红，也不需要硬想——他的道痕感知让他从旁观者视角直接"看到"法则的原貌。但这不是因为他天赋异禀——是因为他的前世已经把所有路都走过一遍了。这一世他只是重新走一遍自己修过的路。他的"躺平"恰好创造了最佳的旁观状态——不动手、不干预、不执迷，法则在他眼里就是透明的。

**子层**：深痕 → 融痕 → 破痕 → 刻骨
- 深痕：形成第一道基于独立理解的自刻道痕
- 融痕：多条自刻道痕开始交织，形成个人道痕区域
- 破痕：理解深到能"破坏"他人刻的浅层道痕（真正开始触及法则边界）
- 刻骨：道痕与神魂骨血融合，为化痕境做准备

### 化痕境（悟道阶段 · 对应传统化神→大乘）

**突破方式**：对法则本身产生颠覆性的新理解。不再是"我想通了一条法则"——是"我看到了一条所有人都在踩但没人注意过的路"。苍渊至今无人到达此境。理论上只有沈溪风到过——他不需要"理解"法则，他直接改写法则。

**法则掌握**：化痕境才真正开始**感悟和掌握法则本身**——不是边界条件，不是具体表现，而是法则的底层脉理。修炼者不再"借用"法则之力，而是成为法则的一部分。

**子层**：道痕 → 化痕 → 归痕
- 道痕：推翻了前人对某条法则的所有理解，建立自己的法则模型。这不是学术突破——是世界观级别的重塑。
- 化痕：将所有独立理解的法则融合为一条"大道"——一条从你这里开始、在你这里结束的完整法则之流。苍渊理论中说这是最高境界，但没人见过。
- 归痕（执痕预备）：道痕与天地法则完全同频，准备执掌天痕之脉本身。

### 执痕（创造阶段 · 超越大乘）

**突破方式**：不只是理解法则——是创造新法则，等同于世界的一部分。苍渊历史上只有沈溪风的前世达到过。他不需要"突破"到执痕——他创造了道痕体系本身。执痕者就是天痕之脉的意志。

**沈安的现状**：前世执痕，现世封印。表面修为被压在窥痕境·初痕（三年未动），实际境界远不止此。每次动用超越刻痕境的力量——道痕共振触发——巡天鉴定位。

## 境界链速查

```
窥痕境(描红)           刻痕境(自痕)              化痕境(悟道)
初痕 → 描痕             深痕 → 融痕                道痕 → 化痕 → 归痕
  → 明痕 → 透痕           → 破痕 → 刻骨            需要颠覆性理解
描够了就突破           必须自己想通一条             法则即我
完全不需要理解法则      初步接触法则边界             真正感悟掌握法则
```

> **法则介入深度**：窥痕境完全不接触法则 → 刻痕境·破痕初步接触法则边界 → 化痕境才真正感悟和掌握法则。这是道痕体系与传统体系最大的区别：它不是从"灵气积累"开始的，是从"模仿"开始的。

每境分初/中/后/圆满四小层。跨越大境比传统体系更难——因为道痕突破需要的是理解而非积累。

## 与传统体系的对比

| 维度 | 传统体系 | 窥痕境(下阶) | 刻痕境(中阶) | 化痕境(上阶) |
|---|---|---|---|---|

## 关键人物与道痕

- **沈溪风(创造者)**：前世用拭痕刻下道痕种子投入苍渊。他是唯一到过执痕的人——他不需要"突破"到执痕，他创造了道痕体系本身。
- **沈安(当前的沈溪风)**：在窥痕境·初痕卡了三年——不是在失败，是从来没用过力。他的突破速度取决于他愿意出多大力，而不是他能不能突破。每次动用超过刻痕境的力量——道痕共鸣→巡天鉴定位。

### 沈安的特殊状态：创造者重修

沈安并非"天生道痕奇才"。他是道痕体系的创造者——这东西从理论到实践全是他前世设计的。但他前世走到顶点靠的不是道痕——是**传统修仙体系**。他在玄霄以传统体系修炼到大乘境界，在巅峰时期用拭痕刻下了道痕体系的第一颗道痕种子投入苍渊。

重生到苍渊之后：
- **传统体系**：修为随肉体重置归零。他不能重走传统体系的路——苍渊灵气稀薄，且传统体系在纪元更迭后会清零，修了也白修。
- **道痕体系**：理解全部保留（他是设计者）。他知道每一条法则的底层脉理、每一道描痕的捷径、每一种突破所需的最小条件。从窥痕境·初痕开始从头修炼——不是学习，是**回忆**。
- **重修极快**：传统修士需要数年摸索的描红路径，他可以在第一天就描对。他的瓶颈不在"会不会"——在"敢不敢"。
- **不敢修太快**：每次突破都会在封印上留下裂痕。裂痕越大，在天痕之脉上产生的道痕共鸣越明显。巡天鉴对此敏感——萧长河会看到。

**这一矛盾是全书的核心张力**：他不是在偷懒——他是在"修"和"藏"之间找一条只有他自己能看见的极窄缝隙。每一步都精确到及格线上——不是因为他懒，是因为及格线以下是浪费重修时间，及格线以上是暴露风险。他在做的不是偷懒——是**最精确的隐匿重修**。
- **纪秋萤(造假事件)**：如果青石学宫真的有道痕体系的窥痕境教授能力——描红——她是能靠自己突破初痕的。不需要造假。是学宫把她放在了需要"天才悟性"的位置上，而她还没有到需要用悟性的阶段。

## 天痕之脉

- 定义：沈溪风前世在苍渊布下的道痕种子网络，是道痕体系的底层架构
- 特性：所有道痕修炼者的描摹/刻痕/悟道行为在天痕之脉中留下不可删除的"代价印记"
- 沈安的特殊性：能感知网络走向和异动枢点（前世执痕的残留能力）
- 道痕共振：动用超越刻痕境的力量→在网络上产生共振波动→巡天鉴捕捉定位
- 关键限制：共振强度与力量使用量成正比；共振后需等待自然消散

## 代价印记

- 定义：每一笔道痕层面的交易在天痕之脉中留下的不可删除的印记
- 特性：不可删除、不可修改、道痕体系天道自然铭记
- 战略意义：沈安在第2卷利用此特性追踪内鬼长老的影阁交易痕迹

## 道契：镜痕

> 道痕体系中第一个应用于"人与人之间关系"的机制。详见 `设定集/核心设定/道契设定.md`。

### 核心原理

- **创造者**：萧长河（约三百年前）——通过巡天鉴观测天痕之脉时发现"镜痕之性"规律
- **镜痕**：每一条道痕都有一道镜像。将镜像的接收方设为人而非天痕之脉本身 → 两人共享同一条道痕
- **签约机制**：双方以道痕之力同时刻下镜痕 → 天痕之脉留下不可删除的代价印记 → 契约由天道自行
- **纪元无关**：天痕之脉不受纪元更迭影响，通过道契锚定的修为在纪元更迭后幸存

### 契约类型

| 类型 | 内容 |
|------|------|
| 资源共享 | 灵气、丹药、功法互通 |
| 感知共享 | 一方的道痕感知可被另一方借用 |
| 修为锚定 | 签约者修为通过天痕之脉形成鼎足而定 |
| 修为共享 | 一方修为向另一方按比例单向分流（牵痕术变形） |

### 护约之法

1. **诚心明晓之约**：签约者道痕波动须显示"完整理解"而非"部分理解"
2. **单方解除备份**：天痕之脉底层嵌入隐藏解除之约——创造者或能读取底层者（沈安）可触发
3. **条款可追溯**：所有道契完整条款签约瞬间被天道自行归档

### 牵痕术（道契的黑暗变形）

玉华阁基于道契根基发展出的修为分流秘法。利用"自愿"定义的可操纵性——诱导目标通过仪式产生友好道痕波动，天地自判/天道自行裁定为自主意志 → 签约成立 → 修为持续单向分流。

详见 `设定集/核心设定/道契设定.md` 及 `设定集/扩展资料/界域势力完整设计.md`（玉华阁章节）。

### 沈安的感知道契能力

前世执痕 + 道痕感知 → 能"看到"活跃道契在网络上的镜痕、识别萧长河本命道痕、解析契约条款、触发诚心明晓之约契中暗约解除背契违约契约。限制：每次干预极耗精力；不能批量操作；干预行为会在天痕之脉中留下痕迹。
