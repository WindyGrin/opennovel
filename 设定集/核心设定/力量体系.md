# 力量体系设定

## 双体系概述

| | 传统体系(练气筑基) | 道痕体系 |
|---|---|---|
| 适用区域 | 玄霄、虚海 | 苍渊 |
| 核心依赖 | 灵气 | 从"描红"到"悟道"的渐进理解 |
| 根基 | 旧纪元法则 | 刻印在神魂上，法则无关 |
| 纪元影响 | 崩塌→清零 | 常驻，不受影响 |

## 道痕体系：从描红到悟道

> 道痕体系不是一上来就要"悟"——那是对新手的不公平。它被设计为一条从模仿到创造的渐进之路。

### 下阶：描红之境（人阶·初痕→明痕）

**突破方式**：跟随师门传承或已有功法中记录的道痕路径，反复描摹、抄录、印入己身。

**不需要理解。** 道痕有"传导性"——前人刻下的痕迹可以被后人临摹，从而在自己身上留下浅层刻印。就像学书法先描红——你不需要懂为什么这一笔要这么写，你只需要把笔顺练熟。

**典型场景**：沈氏家族的《青石经》（残本）就是一套供弟子描摹的道痕路径。云隐宗的入门弟子在最初半年做的就是反复描摹《云隐九诀》的起手式道痕。砥砺宗的底层弟子每天在做的是把宗训上的"天道酬勤"四个字描三百遍——描着描着修为就上去了（然后他们以为是自己够勤奋）。

**突破人阶·明痕**：描够了量，道痕自然累积到一定深度。

### 中阶：刻痕之境（人阶·深痕→地阶·破痕）

**突破方式**：必须在至少一条法则上形成自己的独立理解。不能再靠描——**必须自己刻。**

**原理**：临摹的痕迹是浮在表面的，可以被更强的人抹掉。只有自己刻下去的道痕——基于自己的理解——才会跟自己的神魂融在一起。这一步开始需要"悟"——但这里的"悟"不是大彻大悟，只是"我想通了一件事"。

**举例**：云隐宗的一位长老，他的地阶突破来自一次雷雨天在云雾山脉最高峰上站了一整夜——他想通了"云为什么不会掉下来"。不是理解了整个气象法则——只是理解了一条极小的、关于"上升气流与道痕共振频率"的边界条件。但这一条足够他突破到地阶·刻痕。

**沈安的悖论**：他不需要描红，也不需要硬想——他的道痕感知让他从旁观者视角直接"看到"法则的原貌。但这不是因为他天赋异禀——是因为他的前世已经把所有路都走过一遍了。这一世他只是重新走一遍自己修过的路。他的"躺平"恰好创造了最佳的旁观状态——不动手、不干预、不执迷，法则在他眼里就是透明的。

**突破地阶·破痕**：在一条法则上形成足够深的独立理解，以至于能"破坏"别人刻的浅层道痕。

### 上阶：悟道之境（天阶·道痕→归痕）

**突破方式**：对法则本身产生颠覆性的新理解。不再是"我想通了一条法则"——是"我看到了一条所有人都在踩但没人注意过的路"。苍渊至今无人到达此境。理论上只有沈溪风到过——他不需要"理解"法则，他直接改写法则。

**突破天阶·道痕**：在一条法则上推翻前人所有的理解，建立自己的法则模型。这不是学术突破——是世界观级别的重塑。

**归痕**：将所有独立理解的法则融合为一条"大道"——一条从你这里开始、在你这里结束的完整法则之流。苍渊理论中说这是最高境界，但没人见过。

## 境界链速查

```
人阶(描红)         地阶(刻痕)          天阶(悟道)
初痕 → 明痕        深痕 → 刻痕          道痕 → 化痕 → 归痕
  描够了就突破       → 融痕 → 破痕        需要颠覆性理解
                  必须自己想通一条
                  不能抄了
```

每境分初/中/后/圆满四小层。跨越大阶比传统体系更难——因为道痕突破需要的是理解而非积累。

## 与传统体系的对比

| 维度 | 传统体系 | 道痕体系·下阶 | 道痕体系·中阶 | 道痕体系·上阶 |
|---|---|---|---|---|
| 突破方式 | 积累灵气 | 描摹已有道痕 | 独立理解法则 | 颠覆性新理解 |
| 难易度 | 靠天赋+丹药 | 靠勤奋+抄功 | 靠悟性+阅历 | 靠开创性思维 |
| 寿命 | 大幅延长 | 小幅延长 | 中等延长 | 法则级延长 |
| 纪元后 | ❌ | ✅(浅表) | ✅(深刻) | ✅(融合) |

## 关键人物与道痕

- **沈溪风(创造者)**：前世用拭痕刻下道痕种子投入苍渊。他是唯一到过上阶的人。
- **沈安(当前的沈溪风)**：在人阶·初痕卡了三年——不是在失败，是从来没用过力。他的突破速度取决于他愿意出多大力，而不是他能不能突破。
- **纪秋萤(造假事件)**：如果青石学宫真的有道痕体系的下阶教授能力——描红——她是能靠自己突破初痕的。不需要造假。是学宫把她放在了需要"天才悟性"的位置上，而她还没有到需要用悟性的阶段。
