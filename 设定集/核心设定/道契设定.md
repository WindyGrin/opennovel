# 道契设定

## 概述

**道契**是萧长河结合道痕体系创造的一种法则契约——将双方约定刻印在天痕之脉上，契约一旦成立即在天痕之脉中留下不可删除的**代价印记**，由天痕之脉本身强制执行。

它是道痕体系中第一个（也是唯一一个）应用于"人与人之间关系"的机制——在此之前，道痕只是个人与法则之间的事。

## 技术原理

### 镜痕

天痕之脉中的每一条道痕都存在"镜痕之性"——萧长河在分析巡天鉴的观测所得时发现了这个规律：每一条道痕都可以有一道镜像。如果把镜像的"接收方"设为人而非天痕之脉本身，就可以建立两个人的"共契之痕"。

- **签约方式**：双方以道痕之力同时在网络上刻下一道**镜痕**——两道痕迹互相关联
- **道痕特征**：镜痕会留下萧长河本人的道痕创生之印——这是他作为创造者留下的"创生之印"
- **不可磨灭性**：修改或解除需要双方同时同意——单方面违约会被网络反噬
- **纪元无关性**：天痕之脉本身不受纪元更迭影响，故通过道契锚定的修为一并幸存

### 天道自证（核心安全机制）

天痕之脉判断契约成立的唯一条件：**签约时道痕波动显示自主意志**。

萧长河在原始设计中加入了两条诚心明晓之约：

1. **诚心明晓之约**：签约者的道痕波动必须显示"完整理解"而非"部分理解"——如果有人在不知契约全部条款的情况下签约，该契约在天痕之脉层面无效。

2. **单方解除之约**：萧长河在每个道契的契中暗约中嵌入了一条契中暗约——签约的任何一方可以随时独解契约。这个条款不是"可见的"——它是写入天痕之脉底层的**契中暗约**，只有能读取天痕之脉底层的人（沈溪风/沈安，以及创造者萧长河本人）可以触发。

3. **天地存证不灭**：所有道契的完整条款在签约瞬间被天痕之脉天道自行存证——后续修改（如《牵痕术》对签约仪式的"合乎规矩绕过"）无法删除原始条款。

### 契约类型

| 类型 | 内容 | 示例 |
|------|------|------|
| 资源共享 | 约定灵气、丹药、功法互通 | 道侣互助契约 |
| 感知共享 | 一方的道痕感知可被另一方借用 | 师徒传承契约 |
| 修为锚定 | 签约者的修为通过天痕之脉形成鼎足而定 | 萧长河×苏锦年 |
| 修为共享（牵痕术变形） | 一方修为向另一方单向分流 | 玉华阁《牵痕术》 |

## 萧长河×苏锦年·第一份道契

### 创建背景

- 时间：约三百年前，沈溪风"假死"前不久
- 地点：玄霄·萧氏丹阁
- 动机：苏锦年恐惧纪元更迭后修为归零。萧长河相信"一定有办法"——他通过巡天鉴观测天痕之脉多年，从中提炼出对偶刻痕的方法。
- 契约内容：**修为共享锚定**——纪元更迭后苏锦年的修为不会归零，因为她的修为通过道契锚定在天痕之脉上，而非旧纪元法则。

### 契约原文（刻在一块碎玉上）

> "以此痕为证，吾与锦年共承同一片天劫。纪元不灭我痕，则你我修为不灭。"

### 契中暗约（碎玉背面，苏锦年三百年后才发现）

> "若锦年不愿，此契可随时由她单方解除。长河谨上。"

### 沈溪风的介入

萧长河在签约前请沈溪风看过道契的初稿。沈溪风的反应：
- "你预设了她和你一样理解这份契约。"
- 建议增加"完整理解"验证条款
- 建议增加契中暗约

萧长河没有完全理解沈溪风为什么这么"多疑"——但他照做了。这些建议在三百年后救了云隐宗一个年轻弟子。

## 牵痕术：道契的黑暗变形

### 起源

玉华阁的第一批研究者（约两百年前）从苏锦年秘密传下的道契资料中发现了"修为共享"的可能性。她们将道契根基拆解，保留了签约机制，移除了诚心明晓之约的可见性。

### 运作之法

1. 施术者与目标建立某种表面上的"契约关系"——道侣、同修、师徒等
2. 施术者引导目标完成一个看似无害的"仪式"（共同起誓、互赠信物、共契之痕）
3. 仪式中目标产生的"友好道痕波动"被天痕之脉解读为"自主意志"
4. 契约成立，修为以1%-3竟无一件的比例持续从目标分流到施术者
5. 目标修为下降速度极慢——慢到他以为自己"最近状态不好"

### 条件与限制

| 条件 | 说明 |
|------|------|
| 签约者"自愿" | 天痕之脉的立约之要（但"自愿"可以被诱导） |
| 需道契基础 | 施术者需掌握道契的刻印技术 |
| 修为同源 | 双方必须同为道痕体系修炼者 |
| 单份天限 | 最多分流3竟无一件，过此天限天地必有异象 |
| 不可叠加 | 同一目标不能被多份契约同时分流 |
| 可解除 | 目标主动提交解除申请即可——但需要知道契约的存在 |

### 道德灰色与规矩庇护

- 玉华阁官方说法："互助修炼，合乎规矩合理"
- 大部分男性受害者根本不知道契约的存在
- 知道的人——修为已分流，无法追回
- 提出抗议的人——被玉华阁公关体系反击为"诽谤"
- 镇域司：无从查起。契约合乎规矩，签约自愿，数据波动"正常"
- 镇域司大量女修同时持有玉华阁身份：调查永远在"等待补充材料"

## 沈安的感知道契能力

### 能力来源

前世执痕 + 道痕感知 + 萧长河原始道契的创造者同代身份。

### 能做什么

| 能力项 | 表现 |
|--------|------|
| 道契感知 | 能"看到"活跃道契在天痕之脉上的镜痕 |
| 契约溯源 | 识别道契中的萧长河道痕创生之印 |
| 契约解析 | 读取道契条款（需刻意观察） |
| 契约干预 | 触发诚心明晓之约契中暗约，解除背契违约契约 |
| 契约痕迹追踪 | 追踪修为分流的源头和方向 |

### 不能做什么

- 不能主动创建新道契（封印限制）
- 不能批量解除契约（每次干预极耗精力）
- 不能干预他"看不懂"的契约（必须基于前世已理解的道契根基）
- 干预行为会在天痕之脉上留下痕迹——可能被懂行的人感知

> **沈安的悖论**：他能"看"到苍渊有多少份《牵痕术》契约在运行。但他不能全部解除——不是因为做不到，是因为每解除一份，他在天痕之脉上的存在就更清晰一分。救所有人 = 暴露自己。这个选择题贯穿他对玉华阁的整个态度。
