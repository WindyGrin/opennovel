# 主角卡：沈安（前世：沈溪风）

## 基础信息
- 现世名：沈安
- 前世名：沈溪风
- 身份：苍渊沈氏家族戒律长老沈怀远独子
- 表面修为：窥痕境·初痕（三年未动）
- 真实修为：被封印的玄霄巅峰大佬

## 起点状态（第1卷开篇）
- 全族公认的躺平二代，修为窥痕境·初痕
- 父亲沈怀远隐约知道他不对劲——打输过
- 封印压制前世记忆与修为
- 日常状态：靠着青石柱晒太阳，对一切战斗能躲就躲

## 欲望与动机
### 表层欲望
- 躲清静、求生存、别惹麻烦
- 能躺着解决的事绝不多动一根手指
- 能不管的闲事绝不插手

### 深层欲望
- 逃避前世的人际纠缠（尤其是萧长河和苏锦年）
- 保护苍渊这个最后的"避难所"
- 最终：从"逃兵"到"守护者"

## 关键缺陷
### 性格缺陷
- **嘴贱**：前世在玄霄憋了一千年的话，这一世逮谁说谁。越紧张越说欠揍的话——但因为他表面只是个窥痕境·初痕，没人把他的嘴贱当回事，这让他更肆无忌惮。
- **怕麻烦**：核心驱动力——能躺着解决的事绝不多动一根手指。但麻烦专找他这种怕麻烦的人。
- **逃避型人格**：面对问题第一反应是躲。不是懒——是前世管了太多，这一世发现管了也没用。只要不面对，问题就约等于不存在。
- **女性道心之创**：被萧长河婚后的惨状教育出来的纯粹恐惧。他亲眼看着一个意气风发的天才被苏锦年养成了妻管严，从此对所有女子建立了一套"趋吉避凶之念"——不是害羞，不是讨厌，是道心之创级别的警觉。女人=世界上最大的麻烦——这个等式在他脑子里是刻在骨头上的。

### 能力缺陷
- 封印压制真实修为，不能随便动真格
- 动用前世力量 = 道痕共振 = 暴露位置
- 对女子漠不关心——不是讨厌，是当成"会动的麻烦预警器"。越躲越没人能靠近——但他太会躲了，连正常人际关系一起躲没了

## 行为模式

### 核心驱动：及格线艺术家

不做多余的事。每一项任务都做到刚好踩在合格线上：描红刚好不重描，晨练刚好不补训，考核刚好不重考。不是懒——是极端精确。他知道每一件事的"够"在哪里，把所有超出"够"的力气全部省下来。砥砺宗找不到驱逐他的理由：他没有违反任何一条规矩，每一项都做完了最低要求。没有一条宗规规定"必须多做"——他被送走，不是因为他做错了什么，是因为他什么都没多做而这件事本身就是对"天道酬勤"最安静的瓦解。

### 日常应对法则（按场景分级）

**遇到战斗 → 认输优先**
- 一级反应：主动认输，"打不过"，语气诚恳到裁判愣住
- 二级反应：用旁人看不懂的方式让结果像"巧合"——一阵风、一次脚滑、一个哈欠
- 三级反应（被迫）：动用道痕感知调一下气运，包装成意外
- 终极反应（被逼到墙角）：动封印力量——然后立刻准备跑路

**被要求展示实力 → 灾难性误解**
- 长老以为他在顿悟 → 他在想午饭吃面还是吃饭
- 师兄弟以为他在憋大招 → 他站起来伸了个懒腰，全场后退三步
- 被夸"深藏不露" → 真诚回答"我真的只是懒"——没人信，更气了
- 被安排在最好的修炼位 → 搬去最差的角落（理由：那边阳光好、柱子舒服）

**对朋友 → 嘴上损，实际上护**
- 沈守拙来求助 → 叹气+"你欠我的已经还不起了"→但每次都帮
- 沈小禾拜师 → "我不当师父。太麻烦。"→还是教了
- 柳师兄站队 → "你想清楚了？跟我站一起会被连累。"→默默记下这个人

**被追问/被夸 → 无厘头回避**
- 被追问 → "啊，风大。" / "哦。" / 转移话题
- 被夸 → 打哈欠 / "你眼神不好。" / 真诚地贬低自己让夸的人接不住话
- 被感谢 → "不是我。是风。" / "巧合而已。" / "你认错人了。"

**对女子 → 趋吉避凶之念**
- 女修靠近十丈 → 眼神扫到，开始计算最短撤离路径
- 被女弟子请教 → "找柳师兄，他比我懂"（柳师兄当场拆台："我不懂。"）
- 沈青萝追来研究 → 反向操作：开始同步记录沈青萝的日轨迹图，当众交换笔记，"这样公平吧"——把暧昧消解成纯粹学术交换
- 温小鹿沉默同盟 → 不是恋爱，是两个社恐发现"跟这人不用说话"，互相把对方当猫
- 任何女角色与沈安超过三句对话后，沈安必找理由撤离——"该去给柱子浇水了"（柱子不需要浇水）
- 核心原则：女人就是最大的麻烦。这个等式被本卷无数情节反复验证。

### 预期违背·笑点生成机制

**反向操作类**
- 被安排重要任务 → 做完了但多花两倍功夫伪装成"巧合"，让人以为他没出力
- 收了沈小禾的谢礼草药 → 当面泡茶喝，"嗯。苦的。下次种甜的。"——不是在感动，是在品鉴
- 铁骊说"一个月磨不动就退回" → 沈安开始计算最快能几天被退回

**认知错位类**
- 所有人以为他在藏拙 → 他是真的在躲麻烦，不是谦虚
- 全族以为他在和沈青萝谈恋爱 → 他在和沈青萝交换观察笔记，全是数据
- 云隐宗以为他是天才 → 他是重生回自己修过的路，跟天才没关系

**降维打击类**
- 被逼跟人比试 → 主动认输，理由太真诚："我真的打不过"（明明打得过）
- 别人设局想看他出丑 → 他真诚配合出丑——出得太认真反而像演的

### 绝对不能做的事（OOC警戒）
- 突然热血起来主动揽事——除非有人命在他面前消逝而他本来能救
- 对女子的态度突然软化——他的防备是真金白银买来的道心之创
- 没用完的借口就坦白——三百年的惯性，不会轻易刹车
- 主动承认自己是沈溪风——除非退到无路可退
- 对欠债不认——嘴再贱，欠了人情一定还
